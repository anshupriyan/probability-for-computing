# 03. Classification of States (Section 4.3)

In any Markov chain, not all states are created equal. Some states are easy to reach, some are dead ends that you can never escape once you enter, and some are just temporary rest stops along the way. 

To study the long-term behavior of a system, we must first classify its states. Let's build a clear, intuitive guide to state classification.

---

## 1. The Core Analogy: Road Networks and Dead Ends

> 💡 **Mental Model: Traveling Between Cities**
> Imagine the states in a Markov chain are cities on a map, and the transitions are one-way roads connecting them.
> 
> - **Accessibility**: Can you drive from City $i$ to City $j$? If there is a route (either direct or with layovers), then City $j$ is **accessible** from City $i$.
> - **Communication**: Can you drive from City $i$ to City $j$ *and* drive back from City $j$ to City $i$? If yes, they **communicate**. They belong to the same local travel network.
> - **Equivalence Classes (Countries)**: Communication groups the cities into "countries" (classes). You can travel freely within a country, but once you cross a border into another country, you might not be able to return.
> - **Absorbing States**: A state with no exit roads. Once you enter, you are trapped forever.

## 2. Accessibility and Communication

### Accessibility ($i \to j$)
We say state $j$ is **accessible** from state $i$ (written as $i \to j$) if there is a non-zero probability that the process will eventually enter state $j$ starting from $i$.
* **Formally**: $P_{ij}^n > 0$ for some $n \ge 0$.
* **Why $n \ge 0$ matters**: By definition, $P_{ii}^0 = P(X_0 = i \mid X_0 = i) = 1$. Thus, every state is always accessible from itself.
* **Non-Accessibility**: If state $j$ is not accessible from state $i$, then the probability of ever visiting $j$ starting from $i$ is exactly $0$:
  $$P(\text{ever enter } j \mid X_0 = i) \le \sum_{n=0}^\infty P_{ij}^n = 0$$

### Communication ($i \leftrightarrow j$)
If state $j$ is accessible from state $i$, and state $i$ is accessible from state $j$, the two states **communicate** (written as $i \leftrightarrow j$).

#### 📝 Mathematical Proof: Communication is an Equivalence Relation
To show that communication partitions the state space, we prove it satisfies the three properties of an equivalence relation:
1. **Reflexive**: State $i$ communicates with itself ($i \leftrightarrow i$).
   * *Proof*: $P_{ii}^0 = 1 > 0$. Thus, $i \to i$ is trivially true.
2. **Symmetric**: If $i \leftrightarrow j$, then $j \leftrightarrow i$.
   * *Proof*: This follows directly from the definition: $i \leftrightarrow j \implies (i \to j \text{ and } j \to i) \implies (j \to i \text{ and } i \to j) \implies j \leftrightarrow i$.
3. **Transitive**: If $i \leftrightarrow j$ and $j \leftrightarrow k$, then $i \leftrightarrow k$.
   * *Proof*: Since $i \to j$ and $j \to k$, there exist integers $n, m \ge 0$ such that $P_{ij}^n > 0$ and $P_{jk}^m > 0$.
   * Using the **Chapman-Kolmogorov Equations**, we can lower-bound the $(n+m)$-step transition probability from $i$ to $k$ by summing over all paths, specifically focusing on the path going through $j$:
     $$P_{ik}^{n+m} = \sum_{r=0}^\infty P_{ir}^n P_{rk}^m \ge P_{ij}^n P_{jk}^m > 0$$
   * Since $P_{ik}^{n+m} > 0$, we have $i \to k$. By symmetry, we also have $k \to i$. Thus, $i \leftrightarrow k$. $\blacksquare$

### Communication Classes
Because communication is an equivalence relation, it partitions the state space into disjoint **communication classes**. Any two states in the same class communicate with each other, and states in different classes do not communicate in both directions.

### Irreducible Markov Chains
A Markov chain is called **irreducible** if it consists of a **single communication class**. That is, every state is accessible from every other state.

## 3. Recurrence vs. Transience: The Long-Run Visits

Suppose you start at state $i$. Let $f_i$ be the probability that you will **ever reenter** state $i$:
$$f_i = P(\text{ever reenter } i \mid X_0 = i)$$

Based on $f_i$, we classify state $i$ as either **Recurrent** or **Transient**:

| Property | Recurrent State | Transient State |
| :--- | :--- | :--- |
| **Definition** | $f_i = 1$ | $f_i < 1$ |
| **Number of Visits** | Infinite (reenters again and again) | Finite (eventually leaves and never returns) |
| **Expected Visits** | $\infty$ | $\frac{1}{1 - f_i}$ (Geometric distribution) |

### Proposition 4.1: The Divergence Criteria
A state $i$ is:
$$\text{Recurrent if } \sum_{n=1}^\infty P_{ii}^n = \infty$$
$$\text{Transient if } \sum_{n=1}^\infty P_{ii}^n < \infty$$

#### 📝 Proof of Proposition 4.1
1. Let $I_n$ be an indicator variable:
   $$I_n = \begin{cases} 1 & \text{if } X_n = i \\ 0 & \text{if } X_n \ne i \end{cases}$$
2. The total number of visits to state $i$ is $N = \sum_{n=0}^\infty I_n$.
3. The expected number of visits, starting from $i$, is:
   $$E[N \mid X_0 = i] = E\left[ \sum_{n=0}^\infty I_n \;\middle|\; X_0 = i \right] = \sum_{n=0}^\infty E[I_n \mid X_0 = i] = \sum_{n=0}^\infty P_{ii}^n$$
4. **If $i$ is transient ($f_i < 1$)**: The number of visits follows a geometric distribution where each return is a "success" with probability $f_i$. The probability of visiting exactly $k$ times is $f_i^{k-1}(1 - f_i)$. The expected value of this geometric variable is $\frac{1}{1-f_i}$, which is finite. Hence:
   $$\sum_{n=0}^\infty P_{ii}^n = \frac{1}{1-f_i} < \infty \implies \sum_{n=1}^\infty P_{ii}^n < \infty$$
5. **If $i$ is recurrent ($f_i = 1$)**: The probability of returning is 1, so the process returns infinitely often with probability 1. The expected number of visits is infinite:
   $$\sum_{n=0}^\infty P_{ii}^n = \infty \implies \sum_{n=1}^\infty P_{ii}^n = \infty \quad \blacksquare$$

## 4. Class Properties

A property is called a **class property** if, when one state in a class possesses it, all states in that class must also possess it.

### Recurrence is a Class Property
> 🔔 **Corollary 4.2**: If state $i$ is recurrent, and $i \leftrightarrow j$, then state $j$ is recurrent.

#### 📝 Proof of Corollary 4.2
1. Since $i \leftrightarrow j$, there exist integers $k$ and $m$ such that $P_{ij}^k > 0$ and $P_{ji}^m > 0$.
2. For any integer $n \ge 0$, we can construct a path from $j$ to $j$ in $m + n + k$ steps by going:
   $$\text{State } j \xrightarrow{\text{in } m \text{ steps}} \text{State } i \xrightarrow{\text{in } n \text{ steps}} \text{State } i \xrightarrow{\text{in } k \text{ steps}} \text{State } j$$
3. The probability of this specific path is a lower bound for the total transition probability:
   $$P_{jj}^{m+n+k} \ge P_{ji}^m P_{ii}^n P_{ij}^k$$
4. Summing both sides over all $n$:
   $$\sum_{n=1}^\infty P_{jj}^{m+n+k} \ge P_{ji}^m P_{ij}^k \sum_{n=1}^\infty P_{ii}^n$$
5. Since state $i$ is recurrent, $\sum_{n=1}^\infty P_{ii}^n = \infty$. Since $P_{ji}^m P_{ij}^k > 0$, the right-hand side diverges to infinity.
6. Therefore, $\sum_{s=1}^\infty P_{jj}^s = \infty$, proving that state $j$ is recurrent. $\blacksquare$

### Implications:
* **Transience is a Class Property**: If state $i$ is transient and communicates with $j$, then $j$ must be transient (if $j$ were recurrent, then $i$ would be recurrent, creating a contradiction).
* **Finite State Chains Rule**: *In a finite-state Markov chain, not all states can be transient.*
  * **Proof by Contradiction**: Suppose states are $\{0, 1, \dots, M\}$ and all are transient. Each state is visited only finitely many times. Therefore, after some finite time $T$, no state is ever visited again. But the process must always be in some state, leading to a contradiction!
* **Finite Irreducible Rule**: All states in a finite, irreducible Markov chain are recurrent.

## 5. Detailed Walkthrough Examples

### Example A: Irreducible 3-State Chain (Example 4.14)
* **Transition Matrix**:
  $$P = \begin{bmatrix} 1/2 & 1/2 & 0 \\ 1/2 & 1/4 & 1/4 \\ 0 & 1/3 & 2/3 \end{bmatrix}$$
* **Analysis**:
  - Let's check communication paths:
    - $0 \to 1$ (prob $1/2 > 0$), and $1 \to 0$ (prob $1/2 > 0$). So $0 \leftrightarrow 1$.
    - $1 \to 2$ (prob $1/4 > 0$), and $2 \to 1$ (prob $1/3 > 0$). So $1 \leftrightarrow 2$.
  - By transitivity, $0 \leftrightarrow 2$ (indeed, path is $0 \to 1 \to 2$).
  - **Conclusion**: There is only one class: $\{0, 1, 2\}$. The chain is **irreducible**. Since it is finite and irreducible, all states are **recurrent**.

### Example B: Chain with Absorbing State (Example 4.15)
* **Transition Matrix**:
  $$P = \begin{bmatrix} 1/2 & 1/2 & 0 & 0 \\ 1/2 & 1/2 & 0 & 0 \\ 1/4 & 1/4 & 1/4 & 1/4 \\ 0 & 0 & 0 & 1 \end{bmatrix}$$
* **Analysis**:
  - State 3 is absorbing ($P_{33} = 1$). No other state is accessible from 3. So class is $\{3\}$. It is recurrent.
  - States 0 and 1 communicate with each other ($0 \leftrightarrow 1$). Neither can reach 2 or 3. So class is $\{0, 1\}$. It is recurrent (once inside, you never leave).
  - State 2 can reach 0, 1, and 3, but none of them can reach 2. So class is $\{2\}$. Because it can transition to other classes and never return, it is **transient**.
  - **Conclusion**: The classes are $\{0, 1\}$ (recurrent), $\{2\}$ (transient), and $\{3\}$ (recurrent).

### Example C: Multiple Recurrent and Transient Classes (Example 4.17)
* **Transition Matrix**:
  $$P = \begin{bmatrix} 
  1/2 & 1/2 & 0 & 0 & 0 \\
  1/2 & 1/2 & 0 & 0 & 0 \\
  0 & 0 & 1/2 & 1/2 & 0 \\
  0 & 0 & 1/2 & 1/2 & 0 \\
  1/4 & 1/4 & 0 & 0 & 1/2 
  \end{bmatrix}$$
* **Analysis**:
  - States 0 and 1 communicate: $\{0, 1\}$ is a recurrent class.
  - States 2 and 3 communicate: $\{2, 3\}$ is a recurrent class.
  - State 4 can reach class $\{0, 1\}$ (via $4 \to 0$ or $4 \to 1$ with probability $1/4$), but they cannot return to 4. Thus, state 4 is **transient**.
  - **Conclusion**: Classes are $\{0, 1\}$ (recurrent), $\{2, 3\}$ (recurrent), and $\{4\}$ (transient).

## SECTION - A

### Question 1

#### (i) If $X$ and $Y$ are independent random variables, then $\text{Cov}(X, Y) =$ \_\_\_.
**Solution:**
If $X$ and $Y$ are independent, then:
$$\text{E}[XY] = \text{E}[X]\text{E}[Y]$$
By the definition of covariance:
$$\text{Cov}(X, Y) = \text{E}[XY] - \text{E}[X]\text{E}[Y] = \text{E}[X]\text{E}[Y] - \text{E}[X]\text{E}[Y] = \mathbf{0}$$
**Concepts:** [[03. Covariance and Sums]]

#### (ii) Let $F(1) = 0.5$, $F(2) = 0.4$, $F(3) = 0.7$, and $F(4) = 1.0$. Can $F(x)$ serve as the distribution function of a random variable with range $\{1, 2, 3, 4\}$? State the reason.
**Solution:**
**No.** A cumulative distribution function (CDF) $F(x)$ must be a **non-decreasing function** of $x$, meaning that if $x_1 < x_2$, then $F(x_1) \le F(x_2)$.
Here, $1 < 2$, but:
$$F(1) = 0.5 > F(2) = 0.4$$
Since the function decreases between $x=1$ and $x=2$, it cannot serve as a valid CDF.
**Concepts:** [[01. Introduction to Random Variables and CDF]]

#### (iii) $A$ and $B$ are events such that $\text{P}(A \cup B) = \frac{3}{4}$, $\text{P}(A \cap B) = \frac{1}{4}$, and $\text{P}(A') = \frac{2}{3}$. Find $\text{P}(B)$.
**Solution:**
1. First, find $\text{P}(A)$ from its complement:
   $$\text{P}(A) = 1 - \text{P}(A') = 1 - \frac{2}{3} = \frac{1}{3}$$
2. Use the addition rule of probability:
   $$\text{P}(A \cup B) = \text{P}(A) + \text{P}(B) - \text{P}(A \cap B)$$
3. Substitute the known values:
   $$\frac{3}{4} = \frac{1}{3} + \text{P}(B) - \frac{1}{4}$$
4. Isolate $\text{P}(B)$:
   $$\text{P}(B) = \frac{3}{4} + \frac{1}{4} - \frac{1}{3} = 1 - \frac{1}{3} = \mathbf{\frac{2}{3}}$$
**Concepts:** [[02. Axioms and Properties of Probability]]

#### (iv) If $\text{Cov}(X, Y) = -30$ and the mean of $X$ and $Y$ is $12$ and $8$ respectively, then $\text{E}(XY) =$ \_\_\_.
**Solution:**
Using the covariance formula:
$$\text{Cov}(X, Y) = \text{E}[XY] - \text{E}[X]\text{E}[Y]$$
Substitute the given values ($\text{E}[X] = 12$, $\text{E}[Y] = 8$, $\text{Cov}(X, Y) = -30$):
$$-30 = \text{E}[XY] - (12)(8)$$
$$-30 = \text{E}[XY] - 96$$
$$\text{E}[XY] = -30 + 96 = \mathbf{66}$$
**Concepts:** [[03. Covariance and Sums]]

#### (v) Define almost sure convergence.
**Solution:**
A sequence of random variables $X_1, X_2, \dots$ converges **almost surely** (or with probability 1) to a random variable $X$ (denoted by $X_n \xrightarrow{a.s.} X$) if the probability that the limit of $X_n$ equals $X$ is 1:
$$\text{P}\left(\left\{\omega \in \Omega \,\,\middle|\,\, \lim_{n \to \infty} X_n(\omega) = X(\omega)\right\}\right) = 1$$
**Concepts:** [[05. Limit Theorems and Inequalities]]

#### (vi) Name the discrete distributions for which mean is equal to variance and mean is less than variance.
**Solution:**
1. **Mean is equal to variance ($\text{E}[X] = \text{Var}(X)$):** **Poisson Distribution**, where both are equal to the parameter $\lambda$.
2. **Mean is less than variance ($\text{E}[X] < \text{Var}(X)$):** **Negative Binomial Distribution**. If $X$ is the number of trials to get $r$ successes, $\text{E}[X] = \frac{r}{p}$ and $\text{Var}(X) = \frac{r(1-p)}{p^2}$. Since $0 < p < 1$, we have $\text{Var}(X) = \text{E}[X] \cdot \frac{1-p}{p}$, which is greater than $\text{E}[X]$ when $p < 0.5$.
*(Note: If modeled as the number of failures before $r$ successes: $\text{E}[Y] = \frac{r(1-p)}{p}$ and $\text{Var}(Y) = \frac{r(1-p)}{p^2} = \frac{\text{E}[Y]}{p}$. Since $0 < p < 1$, $\text{Var}(Y) > \text{E}[Y]$ always).*
**Concepts:** [[02. Discrete Random Variables]]

#### (vii) If $X$ is a Poisson variate with m.g.f. $\text{M}_X(t) = e^{3(e^t-1)}$, then $\text{P}(X = 2) =$ \_\_\_.
**Solution:**
The MGF of a Poisson distribution with parameter $\lambda$ is $e^{\lambda(e^t-1)}$.
Matching the formulas:
$$\lambda = 3$$
The Poisson PMF is $p(k) = \frac{e^{-\lambda} \lambda^k}{k!}$. For $k=2$:
$$\text{P}(X=2) = \frac{e^{-3} 3^2}{2!} = \mathbf{\frac{9}{2}e^{-3}} \approx 0.224$$
**Concepts:** [[02. Discrete Random Variables]], [[06. Moments and Moment Generating Functions]]

#### (viii) If $X$ and $Y$ are independent r.v.'s with $\mu_X = 3$, $\mu_Y = 5$, $\sigma_X^2 = 8$, and $\sigma_Y^2 = 12$, then obtain the mean and variance of $U = X + 3Y$.
**Solution:**
1. **Mean of $U$:** By linearity of expectation:
   $$\text{E}[U] = \text{E}[X + 3Y] = \text{E}[X] + 3\text{E}[Y] = 3 + 3(5) = \mathbf{18}$$
2. **Variance of $U$:** Since $X$ and $Y$ are independent:
   $$\text{Var}(U) = \text{Var}(X + 3Y) = \text{Var}(X) + 3^2 \text{Var}(Y) = 8 + 9(12) = 8 + 108 = \mathbf{116}$$
**Concepts:** [[04. Expectation and Variance]], [[03. Covariance and Sums]]

#### (ix) If $A$ and $B$ are mutually exclusive events with $\text{P}(A) = 0.37$ and $\text{P}(B) = 0.44$, then $\text{P}(B') =$ \_\_\_ and $\text{P}(A \cap B') =$ \_\_\_.
**Solution:**
1. **Find $\text{P}(B')$:**
   $$\text{P}(B') = 1 - \text{P}(B) = 1 - 0.44 = \mathbf{0.56}$$
2. **Find $\text{P}(A \cap B')$:**
   Since $A$ and $B$ are mutually exclusive, $A \cap B = \varnothing$. This means $A$ is entirely subsetted inside the complement of $B$ ($A \subseteq B'$).
   Therefore, $A \cap B' = A$, which gives:
   $$\text{P}(A \cap B') = \text{P}(A) = \mathbf{0.37}$$
**Concepts:** [[02. Axioms and Properties of Probability]]

#### (x) What is the smallest value of $k$ in Chebyshev's theorem for which the probability that a random variable will take on a value between $\mu - k\sigma$ and $\mu + k\sigma$ is at least $0.99$?
**Solution:**
Chebyshev's inequality states:
$$\text{P}(|X - \mu| \ge k\sigma) \le \frac{1}{k^2} \implies \text{P}(\mu - k\sigma < X < \mu + k\sigma) \ge 1 - \frac{1}{k^2}$$
We set the lower bound to $0.99$:
$$1 - \frac{1}{k^2} = 0.99 \implies \frac{1}{k^2} = 0.01 \implies k^2 = 100 \implies \mathbf{k = 10}$$
**Concepts:** [[05. Limit Theorems and Inequalities]]

### Question 2

#### (a) State Bayes theorem. A seed packaging company purchases $40\%$ of their bean seeds from supplier A and $60\%$ from supplier B and mixes these seeds together. Bean seeds from supplier A have an $85\%$ germination rate and those from supplier B have a $75\%$ germination rate.
##### (i) Find the probability that a seed selected at random from the mixed seeds will germinate.
##### (ii) Given that a seed germinates, find the probability that the seed was purchased from supplier A.

**Solution:**
* **Bayes' Theorem Statement:** Let $A_1, A_2, \dots, A_n$ be a partition of the sample space $S$ such that $\text{P}(A_i) > 0$. For any event $B$ with $\text{P}(B) > 0$:
  $$\text{P}(A_i \mid B) = \frac{\text{P}(B \mid A_i)\text{P}(A_i)}{\sum_{j=1}^n \text{P}(B \mid A_j)\text{P}(A_j)}$$

Let $A$ be the event the seed is from Supplier A: $\text{P}(A) = 0.40$.
Let $B$ be the event the seed is from Supplier B: $\text{P}(B) = 0.60$.
Let $G$ be the event the seed germinates:
$$\text{P}(G \mid A) = 0.85, \quad \text{P}(G \mid B) = 0.75$$

##### (i) Probability that a randomly selected seed germinates:
Using the Law of Total Probability:
$$\text{P}(G) = \text{P}(G \mid A)\text{P}(A) + \text{P}(G \mid B)\text{P}(B)$$
$$\text{P}(G) = (0.85)(0.40) + (0.75)(0.60) = 0.34 + 0.45 = \mathbf{0.79}$$

##### (ii) Probability it came from Supplier A given it germinates:
Using Bayes' Theorem:
$$\text{P}(A \mid G) = \frac{\text{P}(G \mid A)\text{P}(A)}{\text{P}(G)} = \frac{0.34}{0.79} = \mathbf{\frac{34}{79}} \approx 0.4304$$
**Concepts:** [[05. Bayes' Formula and Law of Total Probability]]

#### (b) A biology professor has two graduate assistants helping her with her research. The probability that the older of the two assistants will be absent on any given day is $0.08$, the probability that the younger of the two will be absent on any given day is $0.05$, and the probability that they will both be absent on any given day is $0.02$. Find the probabilities that:
##### (i) Either or both of the graduate assistants will be absent on any given day;
##### (ii) At least one of the two graduate assistants will not be absent on any given day;
##### (iii) Only one of the two graduate assistants will be absent on any given day.

**Solution:**
Let $O$ be the event the older assistant is absent: $\text{P}(O) = 0.08$.
Let $Y$ be the event the younger assistant is absent: $\text{P}(Y) = 0.05$.
Both absent: $\text{P}(O \cap Y) = 0.02$.

##### (i) Either or both absent ($\text{P}(O \cup Y)$):
Using the addition rule:
$$\text{P}(O \cup Y) = \text{P}(O) + \text{P}(Y) - \text{P}(O \cap Y) = 0.08 + 0.05 - 0.02 = \mathbf{0.11}$$

##### (ii) At least one not absent:
"At least one is not absent" is the complement of "both are absent":
$$\text{P}(\text{at least one present}) = 1 - \text{P}(O \cap Y) = 1 - 0.02 = \mathbf{0.98}$$

##### (iii) Only one is absent:
This corresponds to (Older absent AND Younger present) OR (Older present AND Younger absent):
$$\text{P}(\text{only one absent}) = \text{P}(O \cap Y') + \text{P}(O' \cap Y)$$
$$\text{P}(O \cap Y') = \text{P}(O) - \text{P}(O \cap Y) = 0.08 - 0.02 = 0.06$$
$$\text{P}(O' \cap Y) = \text{P}(Y) - \text{P}(O \cap Y) = 0.05 - 0.02 = 0.03$$
$$\text{P}(\text{only one absent}) = 0.06 + 0.03 = \mathbf{0.09}$$
**Concepts:** [[02. Axioms and Properties of Probability]]

### Question 3

#### (a) Consider the probability density function:
$$f(x) = \begin{cases} \frac{x}{2} & \text{for } 0 < x \le 1 \\ \frac{1}{2} & \text{for } 1 < x \le 2 \\ \frac{3-x}{2} & \text{for } 2 \le x < 3 \\ 0 & \text{elsewhere} \end{cases}$$
##### (i) Show that $f(x)$ is the probability density function of $X$.
##### (ii) Determine the distribution function $F(x)$. Hence evaluate $\text{P}(X > 2.5)$.
##### (iii) Compute $\text{P}(1.5 \le X \le 2.5)$.

**Solution:**

##### (i) Verify PDF:
1. $f(x) \ge 0$ everywhere.
2. The total area under $f(x)$ must equal 1:
   $$\int_{0}^{3} f(x) \, dx = \int_{0}^{1} \frac{x}{2} \, dx + \int_{1}^{2} \frac{1}{2} \, dx + \int_{2}^{3} \frac{3-x}{2} \, dx$$
   $$\int_{0}^{1} \frac{x}{2} \, dx = \left[ \frac{x^2}{4} \right]_0^1 = \frac{1}{4}$$
   $$\int_{1}^{2} \frac{1}{2} \, dx = \left[ \frac{x}{2} \right]_1^2 = 1 - \frac{1}{2} = \frac{1}{2}$$
   $$\int_{2}^{3} \frac{3-x}{2} \, dx = \left[ \frac{3x}{2} - \frac{x^2}{4} \right]_2^3 = \left(\frac{9}{2} - \frac{9}{4}\right) - (3 - 1) = \frac{9}{4} - 2 = \frac{1}{4}$$
   $$\text{Total Area} = \frac{1}{4} + \frac{1}{2} + \frac{1}{4} = \mathbf{1}$$
Hence, $f(x)$ is a valid PDF.

##### (ii) Determine CDF $F(x) = \int_{-\infty}^x f(t) \, dt$:
* **For $x < 0$:** $F(x) = 0$
* **For $0 \le x < 1$:**
  $$F(x) = \int_0^x \frac{t}{2} \, dt = \mathbf{\frac{x^2}{4}}$$
* **For $1 \le x < 2$:**
  $$F(x) = \frac{1}{4} + \int_1^x \frac{1}{2} \, dt = \frac{1}{4} + \frac{x-1}{2} = \mathbf{\frac{2x - 1}{4}}$$
* **For $2 \le x < 3$:**
  $$F(x) = \frac{3}{4} + \int_2^x \frac{3-t}{2} \, dt = \frac{3}{4} + \left[ \frac{3t}{2} - \frac{t^2}{4} \right]_2^x = \frac{3}{4} + \left(\frac{3x}{2} - \frac{x^2}{4} - 2\right) = \mathbf{1 - \frac{(3-x)^2}{4}}$$
* **For $x \ge 3$:** $F(x) = 1$

To evaluate $\text{P}(X > 2.5)$:
$$\text{P}(X > 2.5) = 1 - F(2.5) = 1 - \left(1 - \frac{(3-2.5)^2}{4}\right) = \frac{(0.5)^2}{4} = \frac{0.25}{4} = \mathbf{\frac{1}{16}} = 0.0625$$

##### (iii) Compute $\text{P}(1.5 \le X \le 2.5)$:
$$\text{P}(1.5 \le X \le 2.5) = F(2.5) - F(1.5)$$
$$F(2.5) = 1 - \frac{(3-2.5)^2}{4} = \frac{15}{16}$$
$$F(1.5) = \frac{2(1.5) - 1}{4} = \frac{2}{4} = \frac{8}{16}$$
$$\text{P}(1.5 \le X \le 2.5) = \frac{15}{16} - \frac{8}{16} = \mathbf{\frac{7}{16}} = 0.4375$$
**Concepts:** [[03. Continuous Random Variables]]

#### (b) If $X$ has the cumulative distribution function:
$$F(x) = \begin{cases} 0 & \text{for } x < -1 \\ \frac{1}{4} & \text{for } -1 \le x < 1 \\ \frac{1}{2} & \text{for } 1 \le x < 3 \\ \frac{3}{4} & \text{for } 3 \le x < 5 \\ 1 & \text{for } x \ge 5 \end{cases}$$
##### (i) Obtain p.m.f. of $X$.
##### (ii) Compute $\text{P}(X \le 3)$ and $\text{P}(-0.4 < X < 4)$.
##### (iii) Compute mean and variance of $X$.

**Solution:**

##### (i) Obtain PMF:
The random variable is discrete because $F(x)$ is a step function. The jumps in the CDF represent the probabilities at those points:
* At $x = -1$, jump is $\frac{1}{4} - 0 = 1/4$
* At $x = 1$, jump is $\frac{1}{2} - \frac{1}{4} = 1/4$
* At $x = 3$, jump is $\frac{3}{4} - \frac{1}{2} = 1/4$
* At $x = 5$, jump is $1 - \frac{3}{4} = 1/4$

Thus, the PMF is:
$$p(x) = \begin{cases} \frac{1}{4} & \text{for } x \in \{-1, 1, 3, 5\} \\ 0 & \text{otherwise} \end{cases}$$

##### (ii) Compute probabilities:
* $\text{P}(X \le 3) = F(3) = \mathbf{\frac{3}{4}}$
* $\text{P}(-0.4 < X < 4) = \text{P}(X = 1) + \text{P}(X = 3) = \frac{1}{4} + \frac{1}{4} = \mathbf{\frac{1}{2}}$

##### (iii) Compute Mean and Variance:
* **Mean ($\text{E}[X]$):**
  $$\text{E}[X] = \sum x p(x) = -1(1/4) + 1(1/4) + 3(1/4) + 5(1/4) = \frac{8}{4} = \mathbf{2}$$
* **Variance ($\text{Var}(X)$):**
  $$\text{E}[X^2] = \sum x^2 p(x) = (-1)^2(1/4) + (1)^2(1/4) + (3)^2(1/4) + (5)^2(1/4) = \frac{1 + 1 + 9 + 25}{4} = \frac{36}{4} = 9$$
  $$\text{Var}(X) = \text{E}[X^2] - (\text{E}[X])^2 = 9 - (2)^2 = 9 - 4 = \mathbf{5}$$
**Concepts:** [[01. Introduction to Random Variables and CDF]], [[02. Discrete Random Variables]], [[04. Expectation and Variance]]

### Question 4

#### (a) Given that $X$ has the probability mass function $f(x) = \frac{1}{8}\binom{3}{x}$ for $x = 0, 1, 2, 3$, find the moment generating function of random variable $X$ and hence find its mean and variance.
**Solution:**
This is a Binomial distribution with parameters $n = 3$ and $p = 1/2$.
1. **Find the MGF $M_X(t) = \text{E}[e^{tX}]$:**
   $$M_X(t) = \sum_{x=0}^3 e^{tx} \cdot \frac{1}{8}\binom{3}{x} = \frac{1}{8} \sum_{x=0}^3 \binom{3}{x} (e^t)^x (1)^{3-x}$$
   Using the binomial theorem:
   $$M_X(t) = \frac{1}{8} (1 + e^t)^3 = \mathbf{\left(\frac{1}{2} + \frac{1}{2}e^t\right)^3}$$

2. **Obtain Mean using $M'_X(0)$:**
   $$M'_X(t) = 3\left(\frac{1}{2} + \frac{1}{2}e^t\right)^2 \left(\frac{1}{2}e^t\right) = \frac{3}{2}e^t \left(\frac{1}{2} + \frac{1}{2}e^t\right)^2$$
   $$\text{Mean } \text{E}[X] = M'_X(0) = \frac{3}{2}(1)(1)^2 = \mathbf{1.5}$$

3. **Obtain Variance using $M''_X(0) - [M'_X(0)]^2$:**
   Differentiating $M'_X(t)$ using the product rule:
   $$M''_X(t) = \frac{3}{2}e^t \left(\frac{1}{2} + \frac{1}{2}e^t\right)^2 + 3e^{2t} \left(\frac{1}{2} + \frac{1}{2}e^t\right)$$
   $$M''_X(0) = \frac{3}{2}(1) + 3(1) = 4.5 = \text{E}[X^2]$$
   $$\text{Var}(X) = \text{E}[X^2] - (\text{E}[X])^2 = 4.5 - (1.5)^2 = 4.5 - 2.25 = \mathbf{0.75} \quad (\text{or } 3/4)$$
**Concepts:** [[06. Moments and Moment Generating Functions]], [[04. Expectation and Variance]]

#### (b) Let the probability density function of $Y$ be given by:
$$f(y) = \begin{cases} k(y+1) & \text{for } 2 < y < 4 \\ 0 & \text{elsewhere} \end{cases}$$
##### (i) Obtain the value of $k$.
##### (ii) Find $\text{E}[(3Y+2)^2]$ and $\text{Var}(3Y + 5)$.

**Solution:**

##### (i) Find $k$:
$$\int_{2}^{4} k(y+1) \, dy = 1 \implies k \left[ \frac{y^2}{2} + y \right]_2^4 = 1$$
$$k \left[ (8 + 4) - (2 + 2) \right] = 1 \implies 8k = 1 \implies \mathbf{k = \frac{1}{8}}$$

##### (ii) Find $\text{E}[(3Y+2)^2]$ and $\text{Var}(3Y+5)$:
First, find the raw moments of $Y$:
$$\text{E}[Y] = \int_2^4 y f(y) \, dy = \frac{1}{8} \int_2^4 (y^2 + y) \, dy = \frac{1}{8} \left[ \frac{y^3}{3} + \frac{y^2}{2} \right]_2^4$$
$$\text{E}[Y] = \frac{1}{8} \left( \left(\frac{64}{3} + 8\right) - \left(\frac{8}{3} + 2\right) \right) = \frac{1}{8} \left(\frac{56}{3} + 6\right) = \frac{1}{8} \left(\frac{74}{3}\right) = \frac{37}{12} \approx 3.083$$

$$\text{E}[Y^2] = \int_2^4 y^2 f(y) \, dy = \frac{1}{8} \int_2^4 (y^3 + y^2) \, dy = \frac{1}{8} \left[ \frac{y^4}{4} + \frac{y^3}{3} \right]_2^4$$
$$\text{E}[Y^2] = \frac{1}{8} \left( \left(64 + \frac{64}{3}\right) - \left(4 + \frac{8}{3}\right) \right) = \frac{1}{8} \left(60 + \frac{56}{3}\right) = \frac{1}{8} \left(\frac{236}{3}\right) = \frac{59}{6} \approx 9.833$$

Now, compute the variance:
$$\text{Var}(Y) = \text{E}[Y^2] - (\text{E}[Y])^2 = \frac{59}{6} - \left(\frac{37}{12}\right)^2 = \frac{59}{6} - \frac{1369}{144} = \frac{1416 - 1369}{144} = \frac{47}{144} \approx 0.3264$$

* **Compute $\text{Var}(3Y + 5)$:**
  $$\text{Var}(3Y + 5) = 3^2 \text{Var}(Y) = 9 \left(\frac{47}{144}\right) = \mathbf{\frac{47}{16}} = 2.9375$$

* **Compute $\text{E}[(3Y+2)^2]$:**
  $$\text{E}[(3Y+2)^2] = \text{E}[9Y^2 + 12Y + 4] = 9\text{E}[Y^2] + 12\text{E}[Y] + 4$$
  $$= 9\left(\frac{59}{6}\right) + 12\left(\frac{37}{12}\right) + 4 = \frac{177}{2} + 37 + 4 = 88.5 + 41 = \mathbf{129.5}$$
**Concepts:** [[03. Continuous Random Variables]], [[04. Expectation and Variance]]

### Question 5

#### (a) State Lindeberg-Levy central limit theorem. Let $X_1, X_2, \dots, X_n$ be the sequence of independent random variables with distribution defined as $\text{P}(X_k = \pm k^\alpha) = 1/2$. Examine whether the central limit theorem holds.
**Solution:**
* **Lindeberg-Lévy CLT Statement:** See statement in [[05. Limit Theorems and Inequalities#1. Lindeberg-Lévy Central Limit Theorem (i.i.d. Case)]].

To check if CLT holds for independent but non-identically distributed variables, we check **Lyapunov's Condition** (for $\delta = 2$):
1. **Find expectation and variance:**
   $$\text{E}[X_k] = k^\alpha (1/2) + (-k^\alpha)(1/2) = 0$$
   $$\sigma_k^2 = \text{Var}(X_k) = \text{E}[X_k^2] = (k^\alpha)^2 (1/2) + (-k^\alpha)^2 (1/2) = k^{2\alpha}$$
2. **Find sum of variances $s_n^2$:**
   $$s_n^2 = \sum_{k=1}^n \text{Var}(X_k) = \sum_{k=1}^n k^{2\alpha} \approx \int_0^n x^{2\alpha} \, dx = \frac{n^{2\alpha + 1}}{2\alpha + 1} \quad (\text{for } 2\alpha + 1 > 0)$$
3. **Check Lyapunov condition for 4th moment:**
   $$\text{E}[X_k^4] = (k^\alpha)^4(1/2) + (-k^\alpha)^4(1/2) = k^{4\alpha}$$
   $$\sum_{k=1}^n \text{E}[X_k^4] = \sum_{k=1}^n k^{4\alpha} \approx \frac{n^{4\alpha+1}}{4\alpha+1}$$
4. **Evaluate Lyapunov Limit:**
   $$\lim_{n \to \infty} \frac{1}{s_n^4} \sum_{k=1}^n \text{E}[X_k^4] = \lim_{n \to \infty} \frac{\frac{n^{4\alpha+1}}{4\alpha+1}}{\left(\frac{n^{2\alpha+1}}{2\alpha+1}\right)^2} = \lim_{n \to \infty} \frac{(2\alpha+1)^2}{4\alpha+1} \cdot \frac{n^{4\alpha+1}}{n^{4\alpha+2}} = \lim_{n \to \infty} C \cdot \frac{1}{n} = 0$$
Since this limit converges to 0 for $\alpha > -1/2$, Lyapunov's condition is satisfied.
**Conclusion:** The Central Limit Theorem holds for $\alpha > -1/2$.
**Concepts:** [[05. Limit Theorems and Inequalities]]

#### (b) State Chebyshev's inequality. If $X$ is the number scored in a throw of a die, show that the Chebyshev's inequality gives $\text{P}(|X - \mu| \ge 2.5) \le 0.47$, where $\mu$ is the mean of $X$, while the actual probability is $1/3$.
*(Note: The exam question states "pair of die" but uses the parameters of a single die).*

**Solution:**
* **Chebyshev's Inequality Statement:** Let $X$ be a random variable with mean $\mu$ and variance $\sigma^2$. For any $k > 0$:
  $$\text{P}(|X - \mu| \ge k) \le \frac{\sigma^2}{k^2}$$

For a single fair die:
* Mean $\mu = 3.5$
* Variance $\sigma^2 = \frac{35}{12} \approx 2.917$

We want to find the bound for $k = 2.5$:
$$\text{P}(|X - 3.5| \ge 2.5) \le \frac{35/12}{(2.5)^2} = \frac{35}{12 \times 6.25} = \frac{35}{75} = \frac{7}{15} \approx \mathbf{0.4667} \le 0.47$$

##### Actual Probability:
The event $|X - 3.5| \ge 2.5$ means:
$$X - 3.5 \ge 2.5 \implies X \ge 6 \quad \text{or} \quad X - 3.5 \le -2.5 \implies X \le 1$$
Since $X$ is a die roll, the values satisfying this are $1$ and $6$.
$$\text{P}(X \in \{1, 6\}) = \frac{2}{6} = \mathbf{\frac{1}{3}}$$
**Concepts:** [[05. Limit Theorems and Inequalities]]

### Question 6

#### (a) In the instant lottery with $20\%$ winning tickets, if $X$ is equal to the number of winning tickets out of $8$ tickets purchased, what is the distribution of $X$? Also find the probability of purchasing $2$ winning tickets?
**Solution:**

#### (b) Let $X$ be a r.v. following binomial distribution with parameters $n = 8$ and $p = 0.4$. Derive the m.g.f. of $X$ and hence obtain the m.g.f. of $Y = X - 3$.
**Solution:**
1. **Derive MGF of $X$:**
   $$M_X(t) = \text{E}[e^{tX}] = \sum_{x=0}^8 e^{tx} \binom{8}{x} (0.4)^x (0.6)^{8-x}$$
   $$M_X(t) = \sum_{x=0}^8 \binom{8}{x} (0.4 e^t)^x (0.6)^{8-x} = \mathbf{(0.6 + 0.4e^t)^8}$$
2. **Obtain MGF of $Y = X - 3$:**
   Using the linear transformation property of MGFs ($M_{aX+b}(t) = e^{bt} M_X(at)$):
   $$M_Y(t) = \text{E}[e^{t(X-3)}] = e^{-3t} M_X(t) = \mathbf{e^{-3t} (0.6 + 0.4e^t)^8}$$
**Concepts:** [[06. Moments and Moment Generating Functions]]

#### (c) If $X$ is a Poisson variate with parameter $\lambda$ such that $\text{P}(X = 1) = \text{P}(X = 2)$ then find $\text{P}(X = 3)$.
**Solution:**
For a Poisson distribution, $p(k) = \frac{e^{-\lambda} \lambda^k}{k!}$.
Given $\text{P}(X=1) = \text{P}(X=2)$:
$$e^{-\lambda} \frac{\lambda^1}{1!} = e^{-\lambda} \frac{\lambda^2}{2!}$$
$$\lambda = \frac{\lambda^2}{2} \implies 2\lambda = \lambda^2$$
Since $\lambda > 0$, we divide by $\lambda$:
$$\lambda = 2$$
Now, find $\text{P}(X=3)$:
$$\text{P}(X=3) = \frac{e^{-2} 2^3}{3!} = \frac{8e^{-2}}{6} = \mathbf{\frac{4}{3}e^{-2}} \approx 0.1804$$
**Concepts:** [[02. Discrete Random Variables]]

### Question 7

#### (a) If $X$ is a r.v. having a geometric distribution then show that $\text{P}(Y = t \mid X \ge k) = \text{P}(X = t)$, where $Y = X - k$. Which property of geometric distribution does it depict?
**Solution:**
Let $X \sim \text{Geometric}(p)$, where $\text{P}(X = x) = q^{x-1}p$ and $\text{P}(X \ge x) = q^{x-1}$ (where $q = 1-p$).
We want to evaluate:
$$\text{P}(X - k = t \mid X \ge k) = \text{P}(X = k + t \mid X \ge k)$$
Using the definition of conditional probability:
$$\text{P}(X = k+t \mid X \ge k) = \frac{\text{P}(X = k+t \cap X \ge k)}{\text{P}(X \ge k)}$$
Since $t \ge 1$, the event $\{X = k+t\}$ is a subset of $\{X \ge k\}$. Therefore, the intersection is just $\{X = k+t\}$:
$$\text{P}(X = k+t \mid X \ge k) = \frac{\text{P}(X = k+t)}{\text{P}(X \ge k)} = \frac{q^{k+t-1}p}{q^{k-1}} = q^t p = \text{P}(X = t)$$
**Property depicted:** This is the **Memoryless Property**. It shows that the probability of waiting $t$ additional trials does not depend on the fact that we have already waited $k$ trials.
**Concepts:** [[02. Discrete Random Variables]]

#### (b) The random variables $X_i$ ($i = 1, 2, 3, 4$) are independent and identically distributed with p.d.f.
$$f(x) = \frac{1}{\sqrt{48\pi}} \exp\left(-\frac{(x-2)^2}{48}\right)$$
#### Obtain the distribution of (i) $V = \frac{1}{4}\sum_{i=1}^4 X_i$, and (ii) $W = 2X_1 - 3X_2 + X_3$.

**Solution:**
First, identify the distribution of $X_i$. The PDF is that of a normal distribution $N(\mu, \sigma^2)$ where:
* Mean $\mu = 2$
* Variance $\sigma^2 = 24$ (since $2\sigma^2 = 48 \implies \sigma^2 = 24$)

Since linear combinations of independent normal variables are also normally distributed:

##### (i) Distribution of $V = \frac{1}{4}\sum_{i=1}^4 X_i$ (Sample Mean):
* **Mean of $V$:**
  $$\text{E}[V] = \text{E}\left[\frac{1}{4}\sum_{i=1}^4 X_i\right] = \mu = 2$$
* **Variance of $V$:**
  $$\text{Var}(V) = \text{Var}\left(\frac{1}{4}\sum_{i=1}^4 X_i\right) = \frac{1}{16} \sum_{i=1}^4 \text{Var}(X_i) = \frac{4 \times 24}{16} = 6$$
Hence, $V \sim \mathbf{N(2, 6)}$.

##### (ii) Distribution of $W = 2X_1 - 3X_2 + X_3$:
* **Mean of $W$:**
  $$\text{E}[W] = 2\text{E}[X_1] - 3\text{E}[X_2] + \text{E}[X_3] = 2(2) - 3(2) + 2 = \mathbf{0}$$
* **Variance of $W$:**
  $$\text{Var}(W) = 2^2\text{Var}(X_1) + (-3)^2\text{Var}(X_2) + 1^2\text{Var}(X_3) = 4(24) + 9(24) + 24 = 14(24) = \mathbf{336}$$
Hence, $W \sim \mathbf{N(0, 336)}$.
**Concepts:** [[03. Continuous Random Variables]], [[03. Covariance and Sums]]

### Question 8

#### (a) A random variable $X$ has a Gamma distribution and its p.d.f. is given by:
$$f(x) = \frac{1}{\beta^\alpha \Gamma(\alpha)} x^{\alpha-1} e^{-x/\beta}, \quad \text{for } x > 0$$
#### Prove that the $r$-th moment of $X$ about the origin is:
$$\mu'_r = \frac{\beta^r \Gamma(\alpha + r)}{\Gamma(\alpha)}$$
#### Hence find its mean and variance.

**Solution:**
By definition, the $r$-th raw moment is:
$$\mu'_r = \text{E}[X^r] = \int_{0}^{\infty} x^r f(x) \, dx = \int_{0}^{\infty} x^r \frac{1}{\beta^\alpha \Gamma(\alpha)} x^{\alpha-1} e^{-x/\beta} \, dx$$
$$\mu'_r = \frac{1}{\beta^\alpha \Gamma(\alpha)} \int_{0}^{\infty} x^{\alpha+r-1} e^{-x/\beta} \, dx$$
Let $y = \frac{x}{\beta} \implies x = \beta y \implies dx = \beta dy$:
$$\mu'_r = \frac{1}{\beta^\alpha \Gamma(\alpha)} \int_{0}^{\infty} (\beta y)^{\alpha+r-1} e^{-y} (\beta \, dy)$$
$$\mu'_r = \frac{\beta^{\alpha+r}}{\beta^\alpha \Gamma(\alpha)} \int_{0}^{\infty} y^{\alpha+r-1} e^{-y} \, dy$$
Using the Gamma function definition $\Gamma(z) = \int_0^\infty t^{z-1} e^{-t} \, dt$:
$$\mu'_r = \frac{\beta^r \Gamma(\alpha + r)}{\Gamma(\alpha)} \quad \blacksquare$$

##### Mean ($\mu'_1$):
$$\text{Mean} = \mu'_1 = \frac{\beta^1 \Gamma(\alpha+1)}{\Gamma(\alpha)} = \frac{\beta \cdot \alpha\Gamma(\alpha)}{\Gamma(\alpha)} = \mathbf{\alpha\beta}$$

##### Variance ($\text{Var}(X)$):
$$\mu'_2 = \frac{\beta^2 \Gamma(\alpha+2)}{\Gamma(\alpha)} = \frac{\beta^2 (\alpha+1)\alpha\Gamma(\alpha)}{\Gamma(\alpha)} = \alpha(\alpha+1)\beta^2$$
$$\text{Var}(X) = \mu'_2 - (\mu'_1)^2 = \alpha(\alpha+1)\beta^2 - (\alpha\beta)^2 = \alpha^2\beta^2 + \alpha\beta^2 - \alpha^2\beta^2 = \mathbf{\alpha\beta^2}$$
**Concepts:** [[03. Continuous Random Variables]], [[04. Expectation and Variance]], [[06. Moments and Moment Generating Functions]]

#### (b) If $X$ is Uniformly distributed with mean $2$ and variance $1/3$. Find $\text{P}(X \le 2.5)$.
**Solution:**
For $X \sim \text{Uniform}(a, b)$:
* Mean $\frac{a+b}{2} = 2 \implies a+b = 4$
* Variance $\frac{(b-a)^2}{12} = \frac{1}{3} \implies (b-a)^2 = 4 \implies b-a = 2$ (since $b > a$)

Solving the system:
$$a+b = 4, \quad b-a = 2 \implies a = 1, \quad b = 3$$
The CDF of a Uniform distribution is $F(x) = \frac{x-a}{b-a}$ for $a \le x \le b$.
$$\text{P}(X \le 2.5) = F(2.5) = \frac{2.5 - 1}{3 - 1} = \frac{1.5}{2} = \mathbf{0.75} \quad (\text{or } 3/4)$$
**Concepts:** [[03. Continuous Random Variables]]

#### (c) Define hypergeometric distribution. Find its mean. How does it differ from binomial distribution?
**Solution:**
* **Definition:** See definition in [[02. Discrete Random Variables#F. The Hypergeometric Distribution (Sampling Without Replacement)]].
* **Mean:** $\text{E}[X] = n \frac{M}{N}$
* **Differences from Binomial Distribution:**
  1. **Sampling Type:** Binomial represents sampling **with replacement** (trials are independent, success probability remains constant). Hypergeometric represents sampling **without replacement** (trials are dependent, success probability changes after each draw).
  2. **Population:** Binomial applies to infinite populations (or constant processes). Hypergeometric applies strictly to a finite population of size $N$.

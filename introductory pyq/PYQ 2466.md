## SECTION - A

### Question 1

#### (i) If the mean and variance of $X$ is $10$ and $50$ respectively, then $\text{E}(X^2) =$ \_\_\_.
**Solution:**
Using the variance identity:
$$\text{Var}(X) = \text{E}[X^2] - (\text{E}[X])^2$$
Substitute the given values ($\text{E}[X] = 10$, $\text{Var}(X) = 50$):
$$50 = \text{E}[X^2] - 10^2 \implies \text{E}[X^2] = 50 + 100 = \mathbf{150}$$
**Concepts:** [[04. Expectation and Variance]]

#### (ii) Let $F(1) = 0.3$, $F(2) = 0.5$, $F(3) = 0.8$, and $F(4) = 1.2$. Can $F(x)$ serve as distribution function of a r.v. with range $\{1, 2, 3, 4\}$? State the reason.
**Solution:**
**No.** A cumulative distribution function (CDF) $F(x)$ must satisfy the boundary condition:
$$0 \le F(x) \le 1 \quad \text{for all } x \in \mathbb{R}$$
Here, $F(4) = 1.2$, which is greater than 1. Therefore, it is not a valid CDF.
**Concepts:** [[01. Introduction to Random Variables and CDF]]

#### (iii) $A$ and $B$ are events such that $\text{P}(A \cup B) = \frac{3}{4}$, $\text{P}(A \cap B) = \frac{1}{4}$, and $\text{P}(A') = \frac{2}{3}$. Find $\text{P}(A \cap B')$.
**Solution:**
1. Find $\text{P}(A)$:
   $$\text{P}(A) = 1 - \text{P}(A') = 1 - \frac{2}{3} = \frac{1}{3}$$
2. The event $A \cap B'$ represents elements in $A$ but not in $B$ ($A - B$). We calculate this by subtracting the intersection from $A$:
   $$\text{P}(A \cap B') = \text{P}(A) - \text{P}(A \cap B) = \frac{1}{3} - \frac{1}{4} = \mathbf{\frac{1}{12}} \approx 0.0833$$
**Concepts:** [[02. Axioms and Properties of Probability]]

#### (iv) If $\text{Cov}(X, Y) = 20, \text{E}(X) = 15$ and $\text{E}(Y) = 4$, then $\text{E}(XY) =$ \_\_\_.
**Solution:**
Using the covariance identity:
$$\text{Cov}(X, Y) = \text{E}[XY] - \text{E}[X]\text{E}[Y] \implies 20 = \text{E}[XY] - (15)(4) \implies \text{E}[XY] = 20 + 60 = \mathbf{80}$$
**Concepts:** [[03. Covariance and Sums]]

#### (v) Define convergence in probability.
**Solution:**
A sequence of random variables $X_1, X_2, \dots$ converges **in probability** to a random variable $X$ (denoted by $X_n \xrightarrow{p} X$) if for any $\epsilon > 0$:
$$\lim_{n \to \infty} \text{P}(|X_n - X| \ge \epsilon) = 0 \quad \text{or equivalently} \quad \lim_{n \to \infty} \text{P}(|X_n - X| < \epsilon) = 1$$
**Concepts:** [[05. Limit Theorems and Inequalities]]

#### (vi) Let $X$ be a Normal variate with m.g.f. $\text{M}_X(t) = \exp\left(15t + 18t^2\right)$. Find the mean and variance of $X$.
**Solution:**
The MGF of a normal distribution is $e^{\mu t + \frac{1}{2}\sigma^2 t^2}$.
Matching the parameters:
* **Mean ($\mu$):** $\mu = \mathbf{15}$
* **Variance ($\sigma^2$):** $\frac{1}{2}\sigma^2 = 18 \implies \sigma^2 = \mathbf{36}$
**Concepts:** [[03. Continuous Random Variables]], [[06. Moments and Moment Generating Functions]]

#### (vii) If $X$ is a random variable with m.g.f. $\text{M}_X(t) = \left(\frac{1}{3} + \frac{2}{3}e^t\right)^{10}$, then m.g.f. of $Y = X - 3$ is \_\_\_.
**Solution:**
Using the linear shift property of MGFs ($M_{X+b}(t) = e^{bt} M_X(t)$):
$$M_Y(t) = \text{E}[e^{t(X-3)}] = e^{-3t} M_X(t) = \mathbf{e^{-3t} \left(\frac{1}{3} + \frac{2}{3}e^t\right)^{10}}$$
**Concepts:** [[06. Moments and Moment Generating Functions]]

#### (viii) If $X$ and $Y$ are independent r.v.'s with $\mu_X = 13, \mu_Y = 20, \sigma_X^2 = 49$ and $\sigma_Y^2 = 100$, then $\text{E}(4X + 6Y) =$ \_\_\_\_ and $\text{Var}(4X + 6Y) =$ \_\_\_\_.
**Solution:**
1. **Expected Value:**
   $$\text{E}[4X + 6Y] = 4\text{E}[X] + 6\text{E}[Y] = 4(13) + 6(20) = 52 + 120 = \mathbf{172}$$
2. **Variance:** Since $X$ and $Y$ are independent:
   $$\text{Var}(4X + 6Y) = 4^2 \text{Var}(X) + 6^2 \text{Var}(Y) = 16(49) + 36(100) = 784 + 3600 = \mathbf{4384}$$
**Concepts:** [[04. Expectation and Variance]], [[03. Covariance and Sums]]

#### (ix) If $A$ and $B$ are mutually exclusive events with $\text{P}(A) = 0.37$, and $\text{P}(B) = 0.44$, then $\text{P}(A') =$ \_\_\_\_\_\_\_ and $\text{P}(A \cup B) =$ \_\_\_\_\_\_\_\_\_\_\_.
**Solution:**
1. **Find $\text{P}(A')$:**
   $$\text{P}(A') = 1 - \text{P}(A) = 1 - 0.37 = \mathbf{0.63}$$
2. **Find $\text{P}(A \cup B)$:** Since they are mutually exclusive:
   $$\text{P}(A \cup B) = \text{P}(A) + \text{P}(B) = 0.37 + 0.44 = \mathbf{0.81}$$
**Concepts:** [[02. Axioms and Properties of Probability]]

#### (x) What is the smallest value of $k$ in Chebyshev's inequality for which the probability that a random variable will take on a value between $\mu - k\sigma$ and $\mu + k\sigma$ is at least $0.95$?
**Solution:**
Using Chebyshev's bound:
$$\text{P}(\mu - k\sigma < X < \mu + k\sigma) \ge 1 - \frac{1}{k^2}$$
Set the bound to $0.95$:
$$1 - \frac{1}{k^2} = 0.95 \implies \frac{1}{k^2} = 0.05 \implies k^2 = \frac{1}{0.05} = 20 \implies k = \sqrt{20} \approx \mathbf{4.472}$$
**Concepts:** [[05. Limit Theorems and Inequalities]]

### Question 2

#### (a) State Bayes' theorem. At a hospital's emergency room, patients are classified and $20\%$ of them are critical, $30\%$ are serious and $50\%$ are stable. Of the critical ones, $30\%$ die; of the serious, $10\%$ die; and of the stable, $1\%$ die.
##### (i) Find the probability that a patient selected at random will die.
##### (ii) Given that a patient dies, what is the conditional probability that the patient was classified as critical?

**Solution:**
Let $C, R, S$ represent Critical, Serious, and Stable classifications respectively:
$$\text{P}(C) = 0.20, \quad \text{P}(R) = 0.30, \quad \text{P}(S) = 0.50$$
Let $D$ be the event that a patient dies:
$$\text{P}(D \mid C) = 0.30, \quad \text{P}(D \mid R) = 0.10, \quad \text{P}(D \mid S) = 0.01$$

##### (i) Find $\text{P}(D)$:
Using the Law of Total Probability:
$$\text{P}(D) = \text{P}(D \mid C)\text{P}(C) + \text{P}(D \mid R)\text{P}(R) + \text{P}(D \mid S)\text{P}(S)$$
$$\text{P}(D) = (0.30)(0.20) + (0.10)(0.30) + (0.01)(0.50) = 0.06 + 0.03 + 0.005 = \mathbf{0.095} \quad (9.5\%)$$

##### (ii) Find $\text{P}(C \mid D)$:
Using Bayes' Theorem:
$$\text{P}(C \mid D) = \frac{\text{P}(D \mid C)\text{P}(C)}{\text{P}(D)} = \frac{0.06}{0.095} = \frac{60}{95} = \mathbf{\frac{12}{19}} \approx 0.6316$$
**Concepts:** [[05. Bayes' Formula and Law of Total Probability]]

#### (b) The odds that a book on Statistics will be favourably reviewed by $3$ independent critics are $3$ to $2$, $4$ to $3$ and $2$ to $3$ respectively. What is the probability that of the three reviews:
##### (i) All will be favourable.
##### (ii) Exactly one review will be favourable, and
##### (iii) At least one of the reviews will be favourable.

**Solution:**
Let $A, B, C$ be the events of favorable reviews from the three critics:
* Critic 1: Odds 3 to 2 in favor $\implies \text{P}(A) = \frac{3}{3+2} = \frac{3}{5} = 0.6 \implies \text{P}(A') = \frac{2}{5} = 0.4$
* Critic 2: Odds 4 to 3 in favor $\implies \text{P}(B) = \frac{4}{4+3} = \frac{4}{7} \implies \text{P}(B') = \frac{3}{7}$
* Critic 3: Odds 2 to 3 in favor $\implies \text{P}(C) = \frac{2}{2+3} = \frac{2}{5} = 0.4 \implies \text{P}(C') = \frac{3}{5} = 0.6$

Since the reviews are independent:

##### (i) All three reviews are favorable:
$$\text{P}(A \cap B \cap C) = \text{P}(A)\text{P}(B)\text{P}(C) = \left(\frac{3}{5}\right)\left(\frac{4}{7}\right)\left(\frac{2}{5}\right) = \mathbf{\frac{24}{125}} = 0.192$$

##### (ii) Exactly one review is favorable:
$$\text{P}(\text{Exactly 1}) = \text{P}(A B' C') + \text{P}(A' B C') + \text{P}(A' B' C)$$
$$\text{P}(A B' C') = \left(\frac{3}{5}\right)\left(\frac{3}{7}\right)\left(\frac{3}{5}\right) = \frac{27}{125}$$
$$\text{P}(A' B C') = \left(\frac{2}{5}\right)\left(\frac{4}{7}\right)\left(\frac{3}{5}\right) = \frac{24}{125}$$
$$\text{P}(A' B' C) = \left(\frac{2}{5}\right)\left(\frac{3}{7}\right)\left(\frac{2}{5}\right) = \frac{12}{125}$$
$$\text{P}(\text{Exactly 1}) = \frac{27 + 24 + 12}{125} = \mathbf{\frac{63}{125}} = 0.504$$

##### (iii) At least one review is favorable:
$$\text{P}(\text{At least 1}) = 1 - \text{P}(\text{None favorable}) = 1 - \text{P}(A')\text{P}(B')\text{P}(C')$$
$$\text{P}(\text{None favorable}) = \left(\frac{2}{5}\right)\left(\frac{3}{7}\right)\left(\frac{3}{5}\right) = \frac{18}{125}$$
$$\text{P}(\text{At least 1}) = 1 - \frac{18}{125} = \mathbf{\frac{107}{125}} = 0.856$$
**Concepts:** [[01. Sample Space and Events]], [[04. Independent Events]]

### Question 3

#### (a) Let $X$ be a continuous random variable with p.d.f.
$$f(x) = \begin{cases} kx(1-x) & \text{for } 0 < x \le 1 \\ 0 & \text{elsewhere} \end{cases}$$
##### (i) Determine $k$.
##### (ii) Obtain distribution function $F(x)$. Hence evaluate $\text{P}(X > 0.6)$.
##### (iii) Compute $\text{P}(0.2 \le X \le 0.6)$.

**Solution:**

##### (i) Determine $k$:
$$\int_{0}^{1} k(x - x^2) \, dx = 1 \implies k \left[ \frac{x^2}{2} - \frac{x^3}{3} \right]_0^1 = 1$$
$$k \left( \frac{1}{2} - \frac{1}{3} \right) = 1 \implies \frac{k}{6} = 1 \implies \mathbf{k = 6}$$

##### (ii) Obtain CDF $F(x) = \int_0^x 6(t-t^2) \, dt$:
For $0 \le x \le 1$:
$$F(x) = 6 \left[ \frac{t^2}{2} - \frac{t^3}{3} \right]_0^x = \mathbf{3x^2 - 2x^3}$$
Thus:
$$\text{P}(X > 0.6) = 1 - F(0.6) = 1 - \left(3(0.6)^2 - 2(0.6)^3\right) = 1 - (1.08 - 0.432) = 1 - 0.648 = \mathbf{0.352}$$

##### (iii) Compute $\text{P}(0.2 \le X \le 0.6)$:
$$\text{P}(0.2 \le X \le 0.6) = F(0.6) - F(0.2)$$
$$F(0.2) = 3(0.2)^2 - 2(0.2)^3 = 0.12 - 0.016 = 0.104$$
$$\text{P}(0.2 \le X \le 0.6) = 0.648 - 0.104 = \mathbf{0.544}$$
**Concepts:** [[03. Continuous Random Variables]]

#### (b) Let $X$ be a discrete random variable with distribution function:
$$F(x) = \begin{cases} 0 & \text{for } x < 2 \\ \frac{1}{4} & \text{for } 2 \le x < 4 \\ \frac{1}{2} & \text{for } 4 \le x < 6 \\ \frac{5}{6} & \text{for } 6 \le x < 10 \\ 1 & \text{for } x \ge 10 \end{cases}$$
##### (i) Obtain p.m.f. of $X$.
##### (ii) Compute $\text{P}(X \le 9)$ and $\text{P}(2.5 \le X \le 6.5)$.
##### (iii) Compute mean and variance of $X$.

**Solution:**

##### (i) Obtain PMF:
The jumps in the CDF steps determine the probabilities at the jump points:
* At $x=2$: $p(2) = 1/4 - 0 = \mathbf{1/4}$
* At $x=4$: $p(4) = 1/2 - 1/4 = \mathbf{1/4}$
* At $x=6$: $p(6) = 5/6 - 1/2 = \mathbf{1/3}$
* At $x=10$: $p(10) = 1 - 5/6 = \mathbf{1/6}$

##### (ii) Compute probabilities:
* $\text{P}(X \le 9) = F(9) = \mathbf{\frac{5}{6}}$
* $\text{P}(2.5 \le X \le 6.5) = p(4) + p(6) = \frac{1}{4} + \frac{1}{3} = \mathbf{\frac{7}{12}} \approx 0.5833$

##### (iii) Compute mean and variance:
* **Mean ($\text{E}[X]$):**
  $$\text{E}[X] = \sum x p(x) = 2(1/4) + 4(1/4) + 6(1/3) + 10(1/6) = \frac{1}{2} + 1 + 2 + \frac{5}{3} = \frac{31}{6} \approx \mathbf{5.1667}$$
* **Variance ($\text{Var}(X)$):**
  $$\text{E}[X^2] = \sum x^2 p(x) = 4(1/4) + 16(1/4) + 36(1/3) + 100(1/6) = 1 + 4 + 12 + \frac{50}{3} = \frac{101}{3}$$
  $$\text{Var}(X) = \text{E}[X^2] - (\text{E}[X])^2 = \frac{101}{3} - \left(\frac{31}{6}\right)^2 = \frac{1212 - 961}{36} = \mathbf{\frac{251}{36}} \approx 6.9722$$
**Concepts:** [[01. Introduction to Random Variables and CDF]], [[02. Discrete Random Variables]], [[04. Expectation and Variance]]

### Question 4

#### (a) Find the moment generating function of the random variable $X$ whose probability density function is given by:
$$f(x) = \begin{cases} e^{-x}, & \text{for } x > 0 \\ 0, & \text{elsewhere} \end{cases}$$
#### and use it to find the expression for $\mu'_r$. Hence obtain mean and variance of $X$.
**Solution:**
This is an Exponential distribution with parameter $\lambda = 1$.
1. **Find MGF $M_X(t) = \text{E}[e^{tX}]$:**
   $$M_X(t) = \int_{0}^{\infty} e^{tx} e^{-x} \, dx = \int_{0}^{\infty} e^{-(1-t)x} \, dx$$
   This integral converges for $t < 1$:
   $$M_X(t) = \left[ -\frac{1}{1-t} e^{-(1-t)x} \right]_0^\infty = 0 - \left(-\frac{1}{1-t}\right) = \mathbf{\frac{1}{1-t}} \quad (t < 1)$$

2. **Find raw moment expression $\mu'_r$:**
   Recall the Taylor series expansion for $\frac{1}{1-t}$ when $|t| < 1$:
   $$M_X(t) = 1 + t + t^2 + t^3 + \dots + t^r + \dots = \sum_{r=0}^{\infty} t^r = \sum_{r=0}^{\infty} \frac{t^r}{r!} (r!)$$
   Matching coefficients with the MGF expansion definition $M_X(t) = \sum_{r=0}^{\infty} \frac{t^r}{r!} \mu'_r$:
   $$\mathbf{\mu'_r = r!}$$

3. **Compute Mean and Variance:**
   * **Mean ($\mu'_1$):** $\text{E}[X] = \mu'_1 = 1! = \mathbf{1}$
   * **Variance ($\text{Var}(X)$):**
     $$\text{E}[X^2] = \mu'_2 = 2! = 2 \implies \text{Var}(X) = \text{E}[X^2] - (\text{E}[X])^2 = 2 - 1^2 = \mathbf{1}$$
**Concepts:** [[06. Moments and Moment Generating Functions]], [[03. Continuous Random Variables]]

$$f(x) = \frac{|x-2|}{7} \quad \text{for } x = -1, 0, 1, 3$$
#### Find $\text{E}(X^2 - 5X + 3)$ and $\text{Var}(3X + 5)$.

### Question 5

#### (a) State De-Moivre Laplace Central Limit theorem. Let $X_1, X_2, \dots, X_n$ be the sequence of independent random variables with distribution defined as:
$$\text{P}(X_k = 0) = 1 - k^{-2\alpha}, \quad \text{P}(X_k = \pm k^\alpha) = \frac{1}{2} k^{-2\alpha}$$
#### where $\alpha < 1/2$. Show that the central limit theorem holds.
**Solution:**
We use Lyapunov's Condition to check if CLT holds for this sequence of independent, non-i.i.d. variables:
1. **Find mean and variance of $X_k$:**
   $$\text{E}[X_k] = 0 \quad (\text{by symmetry})$$
   $$\text{Var}(X_k) = \text{E}[X_k^2] = (k^\alpha)^2 \left(\frac{1}{2} k^{-2\alpha}\right) + (-k^\alpha)^2 \left(\frac{1}{2} k^{-2\alpha}\right) + 0^2\left(1 - k^{-2\alpha}\right) = 1$$
2. **Find sum of variances $s_n^2$:**
   $$s_n^2 = \sum_{k=1}^n \text{Var}(X_k) = \sum_{k=1}^n 1 = n$$
3. **Verify Lyapunov's condition for absolute 3rd moment ($\delta = 1$):**
   $$\text{E}[|X_k|^3] = |k^\alpha|^3 \left(\frac{1}{2} k^{-2\alpha}\right) + |-k^\alpha|^3 \left(\frac{1}{2} k^{-2\alpha}\right) = k^{3\alpha} k^{-2\alpha} = k^\alpha$$
   $$\sum_{k=1}^n \text{E}[|X_k|^3] = \sum_{k=1}^n k^\alpha \approx \frac{n^{\alpha+1}}{\alpha+1}$$
4. **Evaluate Lyapunov Limit:**
   $$\lim_{n \to \infty} \frac{1}{s_n^3} \sum_{k=1}^n \text{E}[|X_k|^3] = \lim_{n \to \infty} \frac{\frac{n^{\alpha+1}}{\alpha+1}}{n^{3/2}} = \lim_{n \to \infty} \frac{1}{\alpha+1} n^{\alpha - 1/2}$$
Since $\alpha < 1/2$, the exponent is negative ($\alpha - 1/2 < 0$), meaning:
$$\lim_{n \to \infty} n^{\alpha - 1/2} = 0$$
Lyapunov's condition holds.
**Conclusion:** The Central Limit Theorem holds.
**Concepts:** [[05. Limit Theorems and Inequalities]]

#### (b) State Chebyshev's inequality. For $f(x) = 2^{-x}, x = 1, 2, 3, \dots$ prove that Chebyshev's inequality gives $\text{P}(|X-2| < 2) \ge 0.5$, while the actual probability is $7/8$.
**Solution:**
Here, $X$ follows a Geometric distribution with success parameter $p = 1/2$, so the PMF is $p(x) = (1/2)^x = 2^{-x}$ for $x = 1, 2, \dots$
* Mean $\mu = \frac{1}{p} = 2$
* Variance $\sigma^2 = \frac{1-p}{p^2} = \frac{1/2}{1/4} = 2$

1. **Chebyshev Bound:**
   We want to bound $\text{P}(|X-2| < 2) = 1 - \text{P}(|X-2| \ge 2)$.
   Using Chebyshev's inequality with $k=2$:
   $$\text{P}(|X - 2| \ge 2) \le \frac{\sigma^2}{2^2} = \frac{2}{4} = 0.5$$
   $$\text{P}(|X-2| < 2) \ge 1 - 0.5 = \mathbf{0.5} \quad \blacksquare$$

2. **Actual Probability:**
   The inequality $|X-2| < 2$ expands to:
   $$-2 < X-2 < 2 \implies 0 < X < 4 \implies X \in \{1, 2, 3\}$$
   $$\text{P}(X \in \{1, 2, 3\}) = \text{P}(X=1) + \text{P}(X=2) + \text{P}(X=3) = \frac{1}{2} + \frac{1}{4} + \frac{1}{8} = \mathbf{\frac{7}{8}} \quad \blacksquare$$
**Concepts:** [[05. Limit Theorems and Inequalities]], [[02. Discrete Random Variables]]

#### (c) If $X$ is a binomial variate with $n = 5$ such that $\text{P}(X = 1) = 0.4096$ and $\text{P}(X = 2) = 0.2048$. Find $\text{P}(X = 3)$.
**Solution:**
For $X \sim \text{Binomial}(5, p)$:
$$\text{P}(X=1) = \binom{5}{1} p (1-p)^4 = 5p(1-p)^4 = 0.4096$$
$$\text{P}(X=2) = \binom{5}{2} p^2 (1-p)^3 = 10p^2(1-p)^3 = 0.2048$$
Divide $\text{P}(X=2)$ by $\text{P}(X=1)$:
$$\frac{10p^2(1-p)^3}{5p(1-p)^4} = \frac{0.2048}{0.4096} \implies \frac{2p}{1-p} = 0.5$$
$$4p = 1-p \implies 5p = 1 \implies p = 0.2$$
Now, find $\text{P}(X=3)$:
$$\text{P}(X=3) = \binom{5}{3} (0.2)^3 (0.8)^2 = 10 (0.008)(0.64) = \mathbf{0.0512} \quad (5.12\%)$$

### Question 7

#### (a) Derive the mean and variance of negative binomial distribution with p.m.f.
$$f(x) = \binom{x+r-1}{r-1} p^r q^x, \quad \text{for } x = 0, 1, 2, \dots$$
**Solution:**
This models the Negative Binomial distribution as the number of *failures* $X$ before the $r$-th success.
The MGF of this distribution is:
$$M_X(t) = \left( \frac{p}{1 - q e^t} \right)^r$$
1. **Find Mean:**
   $$M'_X(t) = r\left( \frac{p}{1 - q e^t} \right)^{r-1} \left( \frac{p q e^t}{(1 - q e^t)^2} \right)$$
   $$\text{Mean } \text{E}[X] = M'_X(0) = r(1)^{r-1} \left( \frac{p q}{(1-q)^2} \right) = r \cdot \frac{p q}{p^2} = \mathbf{\frac{rq}{p}}$$

2. **Find Variance:**
   Differentiating again and evaluating at $t=0$ yields $\text{E}[X^2] = \frac{rq(q+rq)}{p^2}$.
   $$\text{Var}(X) = \text{E}[X^2] - (\text{E}[X])^2 = \mathbf{\frac{rq}{p^2}}$$
**Concepts:** [[06. Moments and Moment Generating Functions]], [[02. Discrete Random Variables]], [[04. Expectation and Variance]]

#### (b) The random variables $X_i$ ($i = 1, 2, 3, 4, 5$) are independent and identically distributed with p.d.f.
$$f(x) = \frac{1}{\sqrt{18\pi}} \exp\left(-\frac{(x-1)^2}{18}\right)$$
#### Obtain the distribution of (i) $V = \frac{1}{5}\sum_{i=1}^5 X_i$, and (ii) $W = 3X_1 - X_2 + 2X_3$.
**Solution:**
Identify the distribution: $X_i \sim N(\mu=1, \sigma^2=9)$ (since $2\sigma^2 = 18 \implies \sigma^2 = 9$).

##### (i) Distribution of $V = \frac{1}{5}\sum_{i=1}^5 X_i$:
* $\text{E}[V] = 1$
* $\text{Var}(V) = \frac{\sigma^2}{5} = \frac{9}{5} = 1.8$
Thus, $V \sim \mathbf{N(1, 1.8)}$.

##### (ii) Distribution of $W = 3X_1 - X_2 + 2X_3$:
* $\text{E}[W] = 3(1) - 1 + 2(1) = \mathbf{4}$
* $\text{Var}(W) = 9\text{Var}(X_1) + 1\text{Var}(X_2) + 4\text{Var}(X_3) = (9 + 1 + 4)(9) = 14 \times 9 = \mathbf{126}$
Thus, $W \sim \mathbf{N(4, 126)}$.

### Question 8

#### (a) Let $X$ be a random variable having p.d.f.
$$f(x) = \frac{1}{\text{B}(2,3)} x(1-x)^2, \quad 0 < x < 1$$
#### Find (i) $\text{P}(X \le 0.5)$ and (ii) $\text{P}(0.5 < X < 1)$.
**Solution:**
This is a Beta distribution of the first kind with parameters $a=2, b=3$.
First, calculate the Beta function coefficient:
$$\text{B}(2,3) = \frac{\Gamma(2)\Gamma(3)}{\Gamma(5)} = \frac{1! \times 2!}{4!} = \frac{2}{24} = \frac{1}{12}$$
Hence, the PDF is:
$$f(x) = 12x(1-x)^2 = 12x(1 - 2x + x^2) = 12x - 24x^2 + 12x^3$$

##### (i) Find $\text{P}(X \le 0.5)$:
$$\text{P}(X \le 0.5) = \int_0^{0.5} (12x - 24x^2 + 12x^3) \, dx = \left[ 6x^2 - 8x^3 + 3x^4 \right]_0^{0.5}$$
$$= 6(0.25) - 8(0.125) + 3(0.0625) = 1.5 - 1 + 0.1875 = \mathbf{0.6875} \quad (\text{or } 11/16)$$

##### (ii) Find $\text{P}(0.5 < X < 1)$:
$$\text{P}(0.5 < X < 1) = 1 - \text{P}(X \le 0.5) = 1 - 0.6875 = \mathbf{0.3125} \quad (\text{or } 5/16)$$
**Concepts:** [[03. Continuous Random Variables]]

#### (b) If $X$ is uniformly distributed with $\mu'_1 = 1$ and $\mu_2 = 4/3$ then find $\text{P}(X < 0)$.
**Solution:**
For $X \sim \text{Uniform}(a, b)$:
* Mean $\mu'_1 = \frac{a+b}{2} = 1 \implies a+b = 2$
* Variance $\mu_2 = \frac{(b-a)^2}{12} = \frac{4}{3} \implies (b-a)^2 = 16 \implies b-a = 4$

Solving the system:
$$a = -1, \quad b = 3$$
The probability density is $\frac{1}{3 - (-1)} = \frac{1}{4}$ on $[-1, 3]$.
$$\text{P}(X < 0) = \int_{-1}^{0} \frac{1}{4} \, dx = \frac{0 - (-1)}{4} = \mathbf{0.25}$$

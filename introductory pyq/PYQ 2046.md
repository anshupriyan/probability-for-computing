## SECTION - A

### Question 1

#### (a) If $\text{P}(A) = 0.6$, $\text{P}(B) = 0.3$ and $\text{P}(A \cap \bar{B}) = 0.4$, then find the value of $\text{P}(\bar{B} \mid A)$.
**Solution:**
Using the definition of conditional probability:
$$\text{P}(\bar{B} \mid A) = \frac{\text{P}(A \cap \bar{B})}{\text{P}(A)}$$
Substitute the given values:
$$\text{P}(\bar{B} \mid A) = \frac{0.4}{0.6} = \mathbf{\frac{2}{3}} \approx 0.6667$$
**Concepts:** [[03. Conditional Probability]]

#### (b) If $X$ follows an exponential distribution with parameter $4$, then find $\text{Var}(X)$.
**Solution:**
For $X \sim \text{Exponential}(\lambda = 4)$, the variance is:
$$\text{Var}(X) = \frac{1}{\lambda^2} = \frac{1}{4^2} = \mathbf{\frac{1}{16}} = 0.0625$$
**Concepts:** [[03. Continuous Random Variables]], [[04. Expectation and Variance]]

#### (c) State the conditions under which Poisson distribution is a limiting case of the Binomial distribution.
**Solution:**
A Binomial distribution with parameters $n$ and $p$ approaches a Poisson distribution with parameter $\lambda$ under the following conditions:
1. The number of trials $n$ is very large ($n \to \infty$).
2. The probability of success $p$ is very small ($p \to 0$).
3. The product $np$ remains constant and finite: $np = \lambda$.
**Concepts:** [[02. Discrete Random Variables]], [[05. Limit Theorems and Inequalities]]

#### (d) If $\text{E}(X) = 5$, $\text{E}(Y) = 6$, $\text{Var}(X) = 15$, $\text{Var}(Y) = 25$, then find $\text{E}(3X^2 + 4Y^2)$.
**Solution:**
First, calculate the second moments using $\text{E}[W^2] = \text{Var}(W) + (\text{E}[W])^2$:
$$\text{E}[X^2] = 15 + 5^2 = 40$$
$$\text{E}[Y^2] = 25 + 6^2 = 61$$
By linearity of expectation:
$$\text{E}[3X^2 + 4Y^2] = 3\text{E}[X^2] + 4\text{E}[Y^2] = 3(40) + 4(61) = 120 + 244 = \mathbf{364}$$
**Concepts:** [[04. Expectation and Variance]]

#### (e) Let $X$ be normally distributed with mean $12$ and standard deviation $4$, find $\text{P}(0 \le X \le 12)$.
**Solution:**
Standardize the variables to $Z = \frac{X - 12}{4} \sim N(0, 1)$:
$$\text{P}(0 \le X \le 12) = \text{P}\left(\frac{0 - 12}{4} \le Z \le \frac{12 - 12}{4}\right) = \text{P}(-3 \le Z \le 0)$$
Using the symmetry of the normal curve:
$$\text{P}(-3 \le Z \le 0) = \Phi(0) - \Phi(-3) = 0.5 - (1 - \Phi(3))$$
From normal tables, $\Phi(3) \approx 0.9987$:
$$\text{P}(0 \le X \le 12) = 0.5 - (1 - 0.9987) = 0.5 - 0.0013 = \mathbf{0.4987}$$
**Concepts:** [[03. Continuous Random Variables]]

#### (f) A card is drawn at random from a pack of cards. Find the probability that the card is either a spade or a king?
**Solution:**
Let $S$ be the event the card is a spade, and $K$ be the event it is a king.
* $\text{P}(S) = \frac{13}{52}$ (13 spades in a deck of 52)
* $\text{P}(K) = \frac{4}{52}$ (4 kings in a deck of 52)
* $\text{P}(S \cap K) = \frac{1}{52}$ (1 king of spades)

Using the addition rule:
$$\text{P}(S \cup K) = \text{P}(S) + \text{P}(K) - \text{P}(S \cap K) = \frac{13}{52} + \frac{4}{52} - \frac{1}{52} = \frac{16}{52} = \mathbf{\frac{4}{13}} \approx 0.3077$$
**Concepts:** [[02. Axioms and Properties of Probability]]

#### (g) If $\mu_X = 10, \mu_Y = 20, \sigma_X^2 = 25, \sigma_Y^2 = 36$ and $\text{Cov}(X, Y) = 100$, then find $\text{Var}(2X - 3Y)$.
**Solution:**
Same as Question 1(i) in [[PYQ 1849#(i) If mu X = 10 mu Y = 20 sigma X 2 = 25 sigma Y 2 = 36 and Cov X Y = 100 then what is the value of Var 2X - 3Y]].
$$\text{Var}(2X-3Y) = \mathbf{-776}$$
*(Note: A variance cannot mathematically be negative. The input covariance $\text{Cov}(X,Y) = 100$ violates Cauchy-Schwarz because the maximum possible covariance is $\sigma_X\sigma_Y = 5 \times 6 = 30$. This represents a typographical error in the exam paper).*
**Concepts:** [[03. Covariance and Sums]]

#### (h) Let $f(x) = \frac{x^2}{20}$, for $x = 0, 1, 2, 3, 4$. Determine whether $f(x)$ can serve as the probability distribution of a random variable.
**Solution:**
For $f(x)$ to be a valid probability distribution, the sum of all probabilities must equal 1.
Let's sum $f(x)$ for all $x$:
$$\sum_{x=0}^4 f(x) = f(0) + f(1) + f(2) + f(3) + f(4) = \frac{0^2}{20} + \frac{1^2}{20} + \frac{2^2}{20} + \frac{3^2}{20} + \frac{4^2}{20}$$
$$\text{Sum} = \frac{0 + 1 + 4 + 9 + 16}{20} = \frac{30}{20} = 1.5$$
Since the sum of the probabilities is $1.5 \ne 1$, $f(x)$ **cannot** serve as a valid probability distribution.
**Concepts:** [[02. Discrete Random Variables]]

#### (i) If $X$ and $Y$ are independent random variables with variances $2$ and $3$ respectively, find the variance of $Z = 2X + 4Y$.
**Solution:**
Since $X$ and $Y$ are independent, their covariance is 0.
$$\text{Var}(Z) = \text{Var}(2X + 4Y) = 2^2 \text{Var}(X) + 4^2 \text{Var}(Y) = 4(2) + 16(3) = 8 + 48 = \mathbf{56}$$
**Concepts:** [[04. Expectation and Variance]], [[03. Covariance and Sums]]

#### (j) If $\text{M}_X(t) = \left(0.6 + 0.4e^t\right)^8$ then find $\text{P}(X \le 3)$.
**Solution:**
This is the MGF of a Binomial distribution with parameters $n=8$ and $p=0.4$.
We want to evaluate $\text{P}(X \le 3) = \sum_{k=0}^3 \binom{8}{k} (0.4)^k (0.6)^{8-k}$:
* $\text{P}(X=0) = \binom{8}{0} (0.4)^0 (0.6)^8 = 0.0168$
* $\text{P}(X=1) = \binom{8}{1} (0.4)^1 (0.6)^7 = 8(0.4)(0.0279936) = 0.0896$
* $\text{P}(X=2) = \binom{8}{2} (0.4)^2 (0.6)^6 = 28(0.16)(0.046656) = 0.2090$
* $\text{P}(X=3) = \binom{8}{3} (0.4)^3 (0.6)^5 = 56(0.064)(0.07776) = 0.2787$

$$\text{P}(X \le 3) = 0.0168 + 0.0896 + 0.2090 + 0.2787 = \mathbf{0.5941} \quad (59.41\%)$$
**Concepts:** [[06. Moments and Moment Generating Functions]], [[02. Discrete Random Variables]]

#### (a) Define conditional probability with example. A manufacturer of airplane parts knows from past experience that the probability is $0.80$ that an order will be ready for shipment on time, and it is $0.72$ that an order will be ready for shipment on time and will also be delivered on time. What is the probability that such an order will be delivered on time given that it was ready for shipment on time?
**Solution:**
* **Conditional Probability Definition:** See definition in [[03. Conditional Probability#1. The Core Concept The Restricted Universe]].
  * *Example:* Rolling a die and getting an even number given that the roll is greater than 3.

Let $R$ be the event the order is ready on time: $\text{P}(R) = 0.80$.
Let $D$ be the event the order is delivered on time.
Ready and delivered on time: $\text{P}(D \cap R) = 0.72$.

We want to find $\text{P}(D \mid R)$:
$$\text{P}(D \mid R) = \frac{\text{P}(D \cap R)}{\text{P}(R)} = \frac{0.72}{0.80} = \mathbf{0.90} \quad (90\%)$$
**Concepts:** [[03. Conditional Probability]]

#### (b) A committee of $4$ people is to be appointed from $3$ officers of the production department, $4$ officers of the purchase department, $2$ officers of the sales department and $1$ chartered accountant. Find the probability of forming the committee in the following manners:
##### (i) There must be one from each category.
##### (ii) It should have at least one from the purchase department.
##### (iii) The chartered accountant must be in the committee.

**Solution:**
Total people $= 3 + 4 + 2 + 1 = 10$. We choose a committee of size $r = 4$.
Total possible committees:
$$\text{Total Outcomes} = \binom{10}{4} = \frac{10 \times 9 \times 8 \times 7}{24} = 210$$

##### (i) One from each category:
Since we need exactly 1 person from each of the 4 departments (Production, Purchase, Sales, CA):
$$\text{Favorable Outcomes} = \binom{3}{1}\binom{4}{1}\binom{2}{1}\binom{1}{1} = 3 \times 4 \times 2 \times 1 = 24$$
$$\text{P}(\text{one from each}) = \frac{24}{210} = \mathbf{\frac{4}{35}} \approx 0.1143$$

##### (ii) At least one from the purchase department:
The complement event is "no officers from the purchase department are selected."
Number of non-purchase department officers $= 10 - 4 = 6$.
$$\text{Favorable Outcomes (Complement)} = \binom{6}{4} = 15$$
$$\text{P}(\text{at least one purchase}) = 1 - \frac{15}{210} = 1 - \frac{1}{14} = \mathbf{\frac{13}{14}} \approx 0.9286$$

##### (iii) The chartered accountant must be in the committee:
Since the 1 CA must be in the committee, we only need to choose the remaining 3 officers from the other 9 people:
$$\text{Favorable Outcomes} = \binom{1}{1} \binom{9}{3} = \frac{9 \times 8 \times 7}{6} = 84$$
$$\text{P}(\text{CA in committee}) = \frac{84}{210} = \mathbf{\frac{2}{5}} = 0.40$$
**Concepts:** [[01. Sample Space and Events]]

### Question 3

#### (a) Find the mean and variance of the random variable $X$ which has the following probability density function:
$$f(x) = \begin{cases} x, & 0 < x < 1 \\ 2-x, & 1 \le x < 2 \\ 0, & \text{otherwise} \end{cases}$$
**Solution:**
This is the triangular distribution PDF.
1. **Find Mean $\text{E}[X]$:**
   $$\text{E}[X] = \int_{0}^{2} x f(x) \, dx = \int_0^1 x^2 \, dx + \int_1^2 x(2-x) \, dx$$
   $$\int_0^1 x^2 \, dx = \left[ \frac{x^3}{3} \right]_0^1 = \frac{1}{3}$$
   $$\int_1^2 (2x - x^2) \, dx = \left[ x^2 - \frac{x^3}{3} \right]_1^2 = \left(4 - \frac{8}{3}\right) - \left(1 - \frac{1}{3}\right) = \frac{4}{3} - \frac{2}{3} = \frac{2}{3}$$
   $$\text{Mean } \text{E}[X] = \frac{1}{3} + \frac{2}{3} = \mathbf{1}$$
   *(Note: This matches the symmetry of the PDF about $x=1$).*

2. **Find Variance $\text{Var}(X) = \text{E}[X^2] - (\text{E}[X])^2$:**
   $$\text{E}[X^2] = \int_0^1 x^3 \, dx + \int_1^2 x^2(2-x) \, dx$$
   $$\int_0^1 x^3 \, dx = \left[ \frac{x^4}{4} \right]_0^1 = \frac{1}{4}$$
   $$\int_1^2 (2x^2 - x^3) \, dx = \left[ \frac{2x^3}{3} - \frac{x^4}{4} \right]_1^2 = \left(\frac{16}{3} - 4\right) - \left(\frac{2}{3} - \frac{1}{4}\right) = \frac{4}{3} - \frac{5}{12} = \frac{11}{12}$$
   $$\text{E}[X^2] = \frac{1}{4} + \frac{11}{12} = \frac{14}{12} = \frac{7}{6}$$
   $$\text{Variance } \text{Var}(X) = \frac{7}{6} - 1^2 = \mathbf{\frac{1}{6}} \approx 0.1667$$
**Concepts:** [[03. Continuous Random Variables]], [[04. Expectation and Variance]]

#### (b) Suppose that the time in minutes that a person has to wait at a certain bus stop for a bus is found to be a random phenomenon, with a probability density function given by:
$$f(x) = \begin{cases} 1/8 & \text{for } 0 \le x < 2 \\ x/8 & \text{for } 2 \le x < 4 \\ 0 & \text{otherwise} \end{cases}$$
##### (i) Obtain the c.d.f. of $X$.
##### (ii) What is the probability that a person will have to wait more than two minutes?
##### (iii) Obtain mean of $X$.

**Solution:**

##### (i) Obtain CDF $F(x) = \int_0^x f(t) \, dt$:
* **For $x < 0$:** $F(x) = 0$
* **For $0 \le x < 2$:**
  $$F(x) = \int_0^x \frac{1}{8} \, dt = \mathbf{\frac{x}{8}}$$
* **For $2 \le x < 4$:**
  $$F(x) = F(2) + \int_2^x \frac{t}{8} \, dt = \frac{2}{8} + \left[ \frac{t^2}{16} \right]_2^x = \frac{1}{4} + \frac{x^2 - 4}{16} = \mathbf{\frac{x^2}{16}}$$
* **For $x \ge 4$:** $F(x) = 1$

##### (ii) Probability of waiting more than 2 minutes ($\text{P}(X > 2)$):
$$\text{P}(X > 2) = 1 - F(2) = 1 - \frac{2}{8} = \mathbf{0.75} \quad (75\%)$$

##### (iii) Mean of $X$:
$$\text{E}[X] = \int_0^2 x\left(\frac{1}{8}\right) \, dx + \int_2^4 x\left(\frac{x}{8}\right) \, dx = \left[ \frac{x^2}{16} \right]_0^2 + \left[ \frac{x^3}{24} \right]_2^4$$
$$\text{E}[X] = \frac{4}{16} + \left(\frac{64 - 8}{24}\right) = \frac{1}{4} + \frac{56}{24} = \frac{1}{4} + \frac{7}{3} = \mathbf{\frac{31}{12}} \approx 2.5833 \text{ minutes}$$
**Concepts:** [[03. Continuous Random Variables]]

### Question 6

#### (a) Obtain the Moment Generating Function of the Binomial distribution. Derive from the result that the sum of two binomial variates is a binomial variate if the variates are independent and have the same probability of success.
**Solution:**
1. **Derive Binomial MGF:**
   $$M_X(t) = \text{E}[e^{tX}] = \sum_{x=0}^n e^{tx} \binom{n}{x} p^x q^{n-x} = \sum_{x=0}^n \binom{n}{x} (pe^t)^x q^{n-x} = \mathbf{(q + pe^t)^n}$$

2. **Sum of two Binomials:**
   Let $X_1 \sim \text{Binomial}(n_1, p)$ and $X_2 \sim \text{Binomial}(n_2, p)$ be independent.
   The MGF of $Z = X_1 + X_2$ is:
   $$M_Z(t) = M_{X_1}(t) \cdot M_{X_2}(t) = (q + pe^t)^{n_1} (q + pe^t)^{n_2} = \mathbf{(q + pe^t)^{n_1 + n_2}}$$
   This is the MGF of a Binomial distribution with parameters $n_1 + n_2$ and $p$. By the uniqueness theorem of MGFs:
   $$Z \sim \mathbf{\text{Binomial}(n_1 + n_2, p)}$$
**Concepts:** [[06. Moments and Moment Generating Functions]], [[02. Discrete Random Variables]]

#### (b) If $X$ and $Y$ are independent Poisson variates with parameters $\lambda$ and $\mu$ respectively such that $\text{P}(X=1) = \text{P}(X=2)$ and $\text{P}(Y=2) = \text{P}(Y=3)$. Find the variance of $X - 2Y$.
**Solution:**
1. **Find $\lambda$ from $P(X=1) = P(X=2)$:**
   $$e^{-\lambda} \lambda = e^{-\lambda} \frac{\lambda^2}{2} \implies \lambda = 2$$
2. **Find $\mu$ from $P(Y=2) = P(Y=3)$:**
   $$e^{-\mu} \frac{\mu^2}{2} = e^{-\mu} \frac{\mu^3}{6} \implies \frac{1}{2} = \frac{\mu}{6} \implies \mu = 3$$
3. **Calculate $\text{Var}(X - 2Y)$:** Since $X$ and $Y$ are independent:
   $$\text{Var}(X - 2Y) = \text{Var}(X) + 4\text{Var}(Y)$$
   For Poisson, variance equals the parameter:
   $$\text{Var}(X - 2Y) = \lambda + 4\mu = 2 + 4(3) = \mathbf{14}$$
**Concepts:** [[02. Discrete Random Variables]], [[03. Covariance and Sums]]

### Question 7

#### (a) Prove that a linear combination of independent normal variates is also a normal variate.
**Solution:**
Let $X_i \sim N(\mu_i, \sigma_i^2)$ for $i = 1, \dots, n$ be independent normal variables. Let $Y = \sum_{i=1}^n a_i X_i$.
The MGF of each $X_i$ is $M_{X_i}(t) = e^{\mu_i t + \frac{1}{2}\sigma_i^2 t^2}$.
Find the MGF of $Y$:
$$M_Y(t) = \text{E}\left[ e^{t \sum a_i X_i} \right] = \text{E}\left[ \prod_{i=1}^n e^{(a_i t) X_i} \right]$$
Since $X_i$ are independent:
$$M_Y(t) = \prod_{i=1}^n M_{X_i}(a_i t) = \prod_{i=1}^n e^{\mu_i (a_i t) + \frac{1}{2}\sigma_i^2 (a_i t)^2}$$
$$M_Y(t) = e^{\left(\sum a_i \mu_i\right) t + \frac{1}{2} \left(\sum a_i^2 \sigma_i^2\right) t^2}$$
This is the MGF of a Normal distribution with mean $\sum a_i \mu_i$ and variance $\sum a_i^2 \sigma_i^2$. By the uniqueness theorem of MGFs, $Y$ is normally distributed.
**Concepts:** [[06. Moments and Moment Generating Functions]], [[03. Continuous Random Variables]]

#### (b) Two independent random variables $X$ and $Y$ are both normally distributed with means $1$ and $2$ and standard deviations $3$ and $4$ respectively. If $Z = X - Y$, then:
##### (i) Identify the distribution of $Z$ and write its p.d.f.
##### (ii) State the mean, median and standard deviation of the distribution of $Z$.
##### (iii) Find $\text{P}(Z + 1 \le 0)$.

**Solution:**

##### (i) Identify distribution and PDF:
Since $X \sim N(1, 9)$ and $Y \sim N(2, 16)$ are independent:
* $\text{E}[Z] = \text{E}[X] - \text{E}[Y] = 1 - 2 = -1$
* $\text{Var}(Z) = \text{Var}(X) + \text{Var}(Y) = 9 + 16 = 25 \implies \text{SD}(Z) = 5$

Thus, $Z \sim \mathbf{N(-1, 25)}$.
The PDF of $Z$ is:
$$f(z) = \mathbf{\frac{1}{5\sqrt{2\pi}} e^{-\frac{(z+1)^2}{50}}, \quad -\infty < z < \infty}$$

##### (ii) Parameters of $Z$:
* **Mean:** $\mathbf{-1}$
* **Median:** $\mathbf{-1}$ (since normal distribution is perfectly symmetric around its mean)
* **Standard Deviation:** $\mathbf{5}$

##### (iii) Find $\text{P}(Z + 1 \le 0)$:
$$\text{P}(Z + 1 \le 0) = \text{P}(Z \le -1)$$
Since $-1$ is the mean of $Z$, and the normal curve is symmetric:
$$\text{P}(Z \le -1) = \mathbf{0.5}$$
**Concepts:** [[03. Continuous Random Variables]]

### Question 8

#### (a) Let random variable $X$ denote the kms (in '000 kms) with a certain kind of tyre having the probability density function:
$$f(x) = \begin{cases} \frac{1}{20} e^{-x/20}, & \text{for } x > 0 \\ 0, & \text{for } x \le 0 \end{cases}$$
#### Find the probabilities that one of these tyres will last: (i) at most $10,000$ kms (ii) anywhere from $16,000$ to $24,000$ kms and (iii) at least $30,000$ kms.
**Solution:**
Here, $X$ is in thousands of kms. So $X \sim \text{Exponential}(\lambda = 1/20)$.
The CDF is $F(x) = 1 - e^{-x/20}$ for $x > 0$.

##### (i) At most 10,000 kms ($X \le 10$):
$$\text{P}(X \le 10) = F(10) = 1 - e^{-10/20} = 1 - e^{-0.5} \approx 1 - 0.6065 = \mathbf{0.3935} \quad (39.35\%)$$

##### (ii) Anywhere from 16,000 to 24,000 kms ($16 \le X \le 24$):
$$\text{P}(16 \le X \le 24) = F(24) - F(16) = e^{-16/20} - e^{-24/20} = e^{-0.8} - e^{-1.2}$$
$$\approx 0.4493 - 0.3012 = \mathbf{0.1481} \quad (14.81\%)$$

##### (iii) At least 30,000 kms ($X \ge 30$):
$$\text{P}(X \ge 30) = 1 - F(30) = e^{-30/20} = e^{-1.5} \approx \mathbf{0.2231} \quad (22.31\%)$$
**Concepts:** [[03. Continuous Random Variables]]

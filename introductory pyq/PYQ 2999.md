### Question 1

#### (a) If $\text{P}(A) = 0.3$, $\text{P}(B) = 0.2$ and $\text{P}(A \cap B) = 0.1$, then obtain $\text{P}(\bar{A} \cap \bar{B})$.
**Solution:**
By De Morgan's Law:
$$\bar{A} \cap \bar{B} = (A \cup B)'$$
Using the addition rule:
$$\text{P}(A \cup B) = \text{P}(A) + \text{P}(B) - \text{P}(A \cap B) = 0.3 + 0.2 - 0.1 = 0.4$$
Thus:
$$\text{P}(\bar{A} \cap \bar{B}) = 1 - \text{P}(A \cup B) = 1 - 0.4 = \mathbf{0.6}$$
**Concepts:** [[02. Axioms and Properties of Probability]]

#### (c) If $X$ and $Y$ are independent events and $\text{E}(X) = 10$, $\text{E}(Y) = 20$, then find $\text{E}(XY)$.
**Solution:**
For independent variables:
$$\text{E}[XY] = \text{E}[X]\text{E}[Y] = (10)(20) = \mathbf{200}$$

#### (d) Can, for some random variable $X$, $\text{P}(\mu_X - 2\sigma_X \le X \le \mu_X + 2\sigma_X) = 0.9545$?
**Solution:**
**Yes.** 
By Chebyshev's inequality, for any random variable:
$$\text{P}(|X - \mu| \ge 2\sigma) \le \frac{1}{2^2} = 0.25 \implies \text{P}(|X - \mu| < 2\sigma) \ge 0.75$$
Since $0.9545 \ge 0.75$, this probability satisfies the Chebyshev bound. Specifically, for a **Normal Random Variable** $X \sim N(\mu, \sigma^2)$, the exact probability is $\approx 0.9545$.
**Concepts:** [[05. Limit Theorems and Inequalities]], [[03. Continuous Random Variables]]

#### (e) If $\text{M}_X(t) = \left(0.9 + 0.1e^t\right)^{100}$, then obtain $\text{E}(X^2)$.
**Solution:**
This is the MGF of a Binomial distribution with parameters $n = 100$ and $p = 0.1$.
$$\text{E}[X] = np = 10$$
$$\text{Var}(X) = np(1-p) = 9$$
$$\text{E}[X^2] = \text{Var}(X) + (\text{E}[X])^2 = 9 + 10^2 = \mathbf{109}$$
**Concepts:** [[06. Moments and Moment Generating Functions]], [[02. Discrete Random Variables]]

#### (f) If $\text{E}(X) = 1$, $\text{E}(X^2) = 4$ and $\text{M}_X(t) = e^{\alpha + \beta t + \gamma t^2}$, then obtain the values of $\alpha$, $\beta$ and $\gamma$.
**Solution:**
The MGF $M_X(t) = e^{\alpha + \beta t + \gamma t^2}$ is that of a Normal distribution $N(\mu, \sigma^2)$ where:
1. $M_X(0) = 1 \implies e^\alpha = 1 \implies \mathbf{\alpha = 0}$
2. Comparing $e^{\beta t + \gamma t^2}$ to the standard normal MGF $e^{\mu t + \frac{1}{2}\sigma^2 t^2}$:
   $$\mu = \beta, \quad \sigma^2 = 2\gamma$$
3. Given $\text{E}[X] = \mu = 1 \implies \mathbf{\beta = 1}$
4. Given $\text{E}[X^2] = 4 \implies \text{Var}(X) = \text{E}[X^2] - \text{E}[X]^2 = 4 - 1 = 3$.
   $$\sigma^2 = 3 \implies 2\gamma = 3 \implies \mathbf{\gamma = 1.5}$$
Thus, $\alpha = 0, \beta = 1, \gamma = 1.5$.
**Concepts:** [[06. Moments and Moment Generating Functions]], [[03. Continuous Random Variables]]

#### (g) $X$ and $Y$ are independent normal variates with means $1, 2$ and variances $25$ and $36$ respectively. Compute $\text{E}(e^{\alpha X + \beta Y})$.

#### (h) If $\text{E}(X) = \mu$ and $\text{Var}(X) = \sigma^2$ and using Chebyshev's inequality we get $\text{E}(|X - \mu| \le k\sigma) > 1 - a$, then obtain the value of $a$.

#### (i) If $\sigma_X^2 = 25, \sigma_Y^2 = 36$ and $\text{Cov}(X, Y) = 10$, then obtain $\text{Var}(2X - 3Y)$.
**Solution:**
$$\text{Var}(2X - 3Y) = 4\text{Var}(X) + 9\text{Var}(Y) - 12\text{Cov}(X, Y)$$
$$\text{Var}(2X - 3Y) = 4(25) + 9(36) - 12(10) = 100 + 324 - 120 = \mathbf{304}$$
**Concepts:** [[03. Covariance and Sums]]

#### (b) A coin is tossed three times and the eight possible outcomes are assumed to be equally likely. If $A$ be the event that a head occurs on each of the first two tosses, $B$ is the event that a tail occurs on the third toss and $C$ is the event that exactly two tails in the three tosses, then show that:
##### (i) $A$ and $B$ are independent events.
##### (ii) $B$ and $C$ are dependent events.

**Solution:**
The sample space is $S = \{HHH, HHT, HTH, HTT, THH, THT, TTH, TTT\}$.
* $A = \{HHH, HHT\} \implies \text{P}(A) = 2/8 = 1/4$
* $B = \{HHT, HTT, THT, TTT\} \implies \text{P}(B) = 4/8 = 1/2$
* $C = \{HTT, THT, TTH\} \implies \text{P}(C) = 3/8$

##### (i) Check independence of $A$ and $B$:
$$A \cap B = \{HHT\} \implies \text{P}(A \cap B) = 1/8$$
$$\text{P}(A)\text{P}(B) = \left(\frac{1}{4}\right)\left(\frac{1}{2}\right) = \frac{1}{8}$$
Since $\text{P}(A \cap B) = \text{P}(A)\text{P}(B)$, the events $A$ and $B$ are **independent**.

##### (ii) Check independence of $B$ and $C$:
$$B \cap C = \{HTT, THT\} \implies \text{P}(B \cap C) = 2/8 = 1/4$$
$$\text{P}(B)\text{P}(C) = \left(\frac{1}{2}\right)\left(\frac{3}{8}\right) = \frac{3}{16}$$
Since $\text{P}(B \cap C) = \frac{1}{4} = \frac{4}{16} \neq \frac{3}{16}$, the events $B$ and $C$ are **dependent**.
**Concepts:** [[04. Independent Events]]

#### (b) Find the cumulative distribution function (cdf) of random variable $X$ whose probability density function (pdf) is given by:
$$f(x) = \begin{cases} x, & 0 < x < 1 \\ 2-x, & 1 \le x < 2 \\ 0, & \text{otherwise} \end{cases}$$
#### Also sketch the graph of pdf and cdf.
**Solution:**
To find the CDF $F(x) = \int_{-\infty}^x f(t) \, dt$:
* **For $x < 0$:** $F(x) = 0$
* **For $0 \le x < 1$:**
  $$F(x) = \int_0^x t \, dt = \mathbf{\frac{x^2}{2}}$$
* **For $1 \le x < 2$:**
  $$F(x) = F(1) + \int_1^x (2-t) \, dt = \frac{1}{2} + \left[ 2t - \frac{t^2}{2} \right]_1^x = \frac{1}{2} + \left(2x - \frac{x^2}{2} - \frac{3}{2}\right) = \mathbf{2x - \frac{x^2}{2} - 1}$$
* **For $x \ge 2$:** $F(x) = 1$

##### Sketches:
* **PDF $f(x)$:** Forms a symmetric triangle starting at $(0,0)$, peaking at $(1,1)$, and returning to $(2,0)$.
* **CDF $F(x)$:** An S-shaped curve starting at $(0,0)$ (quadratic growth), transitioning at $(1, 0.5)$, and flattening to $1$ at $(2, 1)$.
**Concepts:** [[03. Continuous Random Variables]]

#### (b) Let $b(x; n, \theta) = \binom{n}{x} \theta^x (1-\theta)^{n-x}$ and $\text{B}(x; n, \theta) = \sum_{k=0}^x b(k; n, \theta)$. Show that:
##### (i) $b(x; n, \theta) = b(n - x; n, 1 - \theta)$
##### (ii) $b(x; n, \theta) = \text{B}(x; n, \theta) - \text{B}(x - 1; n, \theta)$
##### (iii) $b(x; n, \theta) = \text{B}(n - x; n, 1 - \theta) - \text{B}(n - x - 1; n, 1 - \theta)$
##### (iv) $\text{B}(x; n, \theta) = 1 - \text{B}(n - x - 1; n, 1 - \theta)$.

**Solution:**

##### (i) Show $b(x; n, \theta) = b(n - x; n, 1 - \theta)$:
$$b(n - x; n, 1 - \theta) = \binom{n}{n-x} (1-\theta)^{n-x} (1-(1-\theta))^{n-(n-x)} = \binom{n}{x} (1-\theta)^{n-x} \theta^x = b(x; n, \theta) \quad \blacksquare$$

##### (ii) Show $b(x; n, \theta) = \text{B}(x; n, \theta) - \text{B}(x - 1; n, \theta)$:
$$\text{B}(x; n, \theta) - \text{B}(x - 1; n, \theta) = \sum_{k=0}^x b(k; n, \theta) - \sum_{k=0}^{x-1} b(k; n, \theta) = b(x; n, \theta) \quad \blacksquare$$

##### (iii) Show $b(x; n, \theta) = \text{B}(n - x; n, 1 - \theta) - \text{B}(n - x - 1; n, 1 - \theta)$:
By substituting $n-x$ for the variable in property (ii):
$$\text{B}(n - x; n, 1 - \theta) - \text{B}(n - x - 1; n, 1 - \theta) = b(n-x; n, 1-\theta)$$
Using the identity from property (i):
$$b(n-x; n, 1-\theta) = b(x; n, \theta) \quad \blacksquare$$

##### (iv) Show $\text{B}(x; n, \theta) = 1 - \text{B}(n - x - 1; n, 1 - \theta)$:
Let $X \sim \text{Binomial}(n, \theta)$ represent the number of successes, and $Y = n - X \sim \text{Binomial}(n, 1-\theta)$ represent the number of failures.
$$\text{B}(x; n, \theta) = \text{P}(X \le x) = \text{P}(n - Y \le x) = \text{P}(Y \ge n - x) = 1 - \text{P}(Y < n - x)$$
Since $Y$ is discrete:
$$\text{P}(Y < n - x) = \text{P}(Y \le n - x - 1) = \text{B}(n - x - 1; n, 1-\theta)$$
Substituting this back yields:
$$\text{B}(x; n, \theta) = 1 - \text{B}(n - x - 1; n, 1 - \theta) \quad \blacksquare$$
**Concepts:** [[02. Discrete Random Variables]]

### Question 7

#### (a) Show that $\alpha = \mu\left[\frac{\mu(1-\mu)}{\sigma^2} - 1\right]$ and $\beta = (1 - \mu)\left[\frac{\mu(1-\mu)}{\sigma^2} - 1\right]$, where $(\alpha, \beta)$ are two parameters of the beta type-I distribution and $(\mu, \sigma^2)$ are the mean and variance of the distribution respectively.
**Solution:**
For a Beta type-I distribution:
$$\mu = \frac{\alpha}{\alpha + \beta} \quad \text{and} \quad \sigma^2 = \frac{\alpha\beta}{(\alpha+\beta)^2(\alpha+\beta+1)}$$
Let $S = \alpha + \beta$. We can write $\alpha$ and $\beta$ in terms of $S$ and $\mu$:
$$\alpha = \mu S, \quad \beta = (1-\mu)S$$
Substitute these into the variance equation:
$$\sigma^2 = \frac{(\mu S)((1-\mu)S)}{S^2(S+1)} = \frac{\mu(1-\mu)S^2}{S^2(S+1)} = \frac{\mu(1-\mu)}{S+1}$$
Solve for $S$:
$$S+1 = \frac{\mu(1-\mu)}{\sigma^2} \implies S = \frac{\mu(1-\mu)}{\sigma^2} - 1$$
Substitute $S$ back into the expressions for $\alpha$ and $\beta$:
$$\alpha = \mu \left[ \frac{\mu(1-\mu)}{\sigma^2} - 1 \right] \quad \blacksquare$$
$$\beta = (1-\mu) \left[ \frac{\mu(1-\mu)}{\sigma^2} - 1 \right] \quad \blacksquare$$
**Concepts:** [[03. Continuous Random Variables]], [[04. Expectation and Variance]]

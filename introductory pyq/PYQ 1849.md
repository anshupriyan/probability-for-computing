## SECTION - A

### Question 1

#### (a) Let $f(x) = c\left(\frac{1}{4}\right)^x$ for $x = 1, 2, 3, \dots$ then the value of $c$ for which $f(x)$ can serve as a probability mass function is \_\_\_.
**Solution:**
For $f(x)$ to be a valid PMF, the sum of all probabilities must equal 1:
$$\sum_{x=1}^{\infty} f(x) = 1 \implies \sum_{x=1}^{\infty} c\left(\frac{1}{4}\right)^x = 1$$
This is an infinite geometric series with first term $a = \frac{c}{4}$ and common ratio $r = \frac{1}{4}$:
$$\frac{\frac{c}{4}}{1 - \frac{1}{4}} = 1 \implies \frac{\frac{c}{4}}{\frac{3}{4}} = 1 \implies \frac{c}{3} = 1 \implies \mathbf{c = 3}$$
**Concepts:** [[02. Discrete Random Variables]]

#### (b) Define simple and composite events with examples.
**Solution:**
1. **Simple Event (Elementary Event):** An event that contains exactly one sample point (outcome) from the sample space.
   * *Example:* Rolling a six-sided die and getting a 6. Event $E = \{6\}$.
2. **Composite Event (Compound Event):** An event that contains two or more sample points from the sample space.
   * *Example:* Rolling a six-sided die and getting an even number. Event $E = \{2, 4, 6\}$.
**Concepts:** [[01. Sample Space and Events]]

#### (c) If $X$ and $Y$ are independent events and $\text{E}(X) = 10, \text{E}(Y) = 20$, then the value of $\text{E}(XY)$ is \_\_\_.
**Solution:**
For any two independent random variables:
$$\text{E}(XY) = \text{E}(X)\text{E}(Y)$$
Substituting the given values:
$$\text{E}(XY) = (10)(20) = \mathbf{200}$$
**Concepts:** [[04. Expectation and Variance]], [[02. Independent Random Variables]]

#### (d) Name the distribution which exhibits lack of memory.
**Solution:**
* **Discrete Case:** **Geometric Distribution**.
* **Continuous Case:** **Exponential Distribution**.
**Concepts:** [[02. Discrete Random Variables]], [[03. Continuous Random Variables]]

#### (e) If $\text{M}_X(t) = \left(0.9 + 0.1 e^t\right)^{100}$, then the value of $\text{E}(X^2)$ is \_\_\_.
**Solution:**
This is the MGF of a Binomial distribution with parameters $n = 100$ and $p = 0.1$.
* Mean $\text{E}[X] = np = 100(0.1) = 10$
* Variance $\text{Var}(X) = np(1-p) = 100(0.1)(0.9) = 9$

Using the variance identity:
$$\text{E}[X^2] = \text{Var}(X) + (\text{E}[X])^2 = 9 + 10^2 = \mathbf{109}$$
*(Alternatively, differentiate $\text{M}_X(t)$ twice and evaluate at $t=0$).*
**Concepts:** [[06. Moments and Moment Generating Functions]], [[02. Discrete Random Variables]]

#### (f) If $X \sim \text{Poisson}(4)$ and $Y = X - 4$, then $\text{M}_Y(t) =$ \_\_\_.
**Solution:**
The MGF of $X$ is $M_X(t) = e^{4(e^t - 1)}$.
Using the linear transformation property of MGFs ($M_{aX+b}(t) = e^{bt} M_X(at)$):
$$M_Y(t) = \text{E}[e^{t(X-4)}] = e^{-4t} M_X(t) = e^{-4t} e^{4(e^t - 1)} = \mathbf{e^{4(e^t - t - 1)}}$$
**Concepts:** [[06. Moments and Moment Generating Functions]], [[02. Discrete Random Variables]]

#### (g) $X$ and $Y$ are independent normal variates with means $1, 2$ and variances $25$ and $36$ respectively. Compute $\text{E}(e^{\alpha X + \beta Y})$.
**Solution:**
This is equivalent to finding the MGF of $W = \alpha X + \beta Y$ evaluated at $t=1$.
Since $X$ and $Y$ are independent normal variables, $W$ is also normally distributed:
* $\text{E}[W] = \alpha \text{E}[X] + \beta \text{E}[Y] = \alpha(1) + \beta(2) = \alpha + 2\beta$
* $\text{Var}(W) = \alpha^2 \text{Var}(X) + \beta^2 \text{Var}(Y) = 25\alpha^2 + 36\beta^2$

The MGF of a normal variable $W \sim N(\mu_W, \sigma_W^2)$ is $M_W(t) = e^{\mu_W t + \frac{1}{2}\sigma_W^2 t^2}$. Evaluating at $t=1$:
$$\text{E}[e^{\alpha X + \beta Y}] = \text{E}[e^{1 \cdot W}] = M_W(1) = \mathbf{e^{(\alpha + 2\beta) + \frac{1}{2}(25\alpha^2 + 36\beta^2)}} = \mathbf{e^{\alpha + 2\beta + 12.5\alpha^2 + 18\beta^2}}$$
**Concepts:** [[03. Continuous Random Variables]], [[06. Moments and Moment Generating Functions]]

#### (h) If $\text{E}(X) = \mu$ and $\text{Var}(X) = \sigma^2$ and using Chebyshev's inequality we get $\text{E}(|X - \mu| \le k\sigma) > 1 - a$, then the value of $a$ is \_\_\_.
**Solution:**
Chebyshev's inequality states:
$$\text{P}(|X - \mu| \ge k\sigma) \le \frac{1}{k^2} \implies \text{P}(|X - \mu| < k\sigma) \ge 1 - \frac{1}{k^2}$$
Matching this with the inequality $\text{P}(|X - \mu| \le k\sigma) > 1 - a$, we get:
$$\mathbf{a = \frac{1}{k^2}}$$
**Concepts:** [[05. Limit Theorems and Inequalities]]

#### (i) If $\mu_X = 10, \mu_Y = 20, \sigma_X^2 = 25, \sigma_Y^2 = 36$ and $\text{Cov}(X, Y) = 100$, then what is the value of $\text{Var}(2X - 3Y)$?
**Solution:**
Using the variance of a linear combination:
$$\text{Var}(2X - 3Y) = 4\text{Var}(X) + 9\text{Var}(Y) - 12\text{Cov}(X, Y)$$
Substituting the values:
$$\text{Var}(2X - 3Y) = 4(25) + 9(36) - 12(100) = 100 + 324 - 1200 = \mathbf{-776}$$
*(Note: A variance cannot mathematically be negative. The input covariance $\text{Cov}(X,Y) = 100$ violates Cauchy-Schwarz because the maximum possible covariance is $\sigma_X\sigma_Y = 5 \times 6 = 30$. This represents a typographical error in the exam paper. The calculation shown above follows the paper's literal numbers).*
**Concepts:** [[03. Covariance and Sums]]

#### (j) State De Moivre Laplace central limit theorem.
**Solution:**
See statement in [[05. Limit Theorems and Inequalities#2. De Moivre–Laplace Central Limit Theorem (Binomial Case)]].

### Question 2

#### (a) A box contains $20$ fuses, of which $5$ are defective. If three of the fuses are selected at random and are removed from the box in succession, what is the probability that all are defective?
**Solution:**
Let $D_1, D_2, D_3$ be the events of drawing a defective fuse on the 1st, 2nd, and 3rd draws respectively.
Using conditional probability:
$$\text{P}(D_1 \cap D_2 \cap D_3) = \text{P}(D_1) \cdot \text{P}(D_2 \mid D_1) \cdot \text{P}(D_3 \mid D_1 \cap D_2)$$
* $\text{P}(D_1) = \frac{5}{20} = \frac{1}{4}$
* $\text{P}(D_2 \mid D_1) = \frac{4}{19}$ (4 defectives left out of 19 total)
* $\text{P}(D_3 \mid D_1 \cap D_2) = \frac{3}{18} = \frac{1}{6}$ (3 defectives left out of 18 total)

$$\text{P}(\text{all defective}) = \frac{1}{4} \cdot \frac{4}{19} \cdot \frac{1}{6} = \mathbf{\frac{1}{114}} \approx 0.0088$$
*(Alternatively, using combinations: $\frac{\binom{5}{3}}{\binom{20}{3}} = \frac{10}{1140} = \frac{1}{114}$).*
**Concepts:** [[03. Conditional Probability]]

#### (b) Let $X$ be a discrete random variable having the probability mass function:
$$f(x) = \frac{|x-2|}{7} \quad \text{for } x = -1, 0, 1, 3$$
#### Obtain the moment generating function of $X$. Hence obtain the mean and variance of $X$.
**Solution:**
1. **Find MGF $M_X(t) = \text{E}[e^{tX}]$:**
   Calculate the probabilities for each $x$:
   * $p(-1) = \frac{|-1-2|}{7} = \frac{3}{7}$
   * $p(0) = \frac{|0-2|}{7} = \frac{2}{7}$
   * $p(1) = \frac{|1-2|}{7} = \frac{1}{7}$
   * $p(3) = \frac{|3-2|}{7} = \frac{1}{7}$
   *(Sum of probabilities is $\frac{3+2+1+1}{7} = 1$).*
   $$M_X(t) = \sum e^{tx} p(x) = \mathbf{\frac{3}{7}e^{-t} + \frac{2}{7} + \frac{1}{7}e^t + \frac{1}{7}e^{3t}}$$

2. **Obtain Mean $\text{E}[X] = M'_X(0)$:**
   $$M'_X(t) = -\frac{3}{7}e^{-t} + \frac{1}{7}e^t + \frac{3}{7}e^{3t}$$
   $$\text{Mean } \text{E}[X] = M'_X(0) = -\frac{3}{7} + \frac{1}{7} + \frac{3}{7} = \mathbf{\frac{1}{7}} \approx 0.1429$$

3. **Obtain Variance $\text{Var}(X) = M''_X(0) - [M'_X(0)]^2$:**
   $$M''_X(t) = \frac{3}{7}e^{-t} + \frac{1}{7}e^t + \frac{9}{7}e^{3t}$$
   $$\text{E}[X^2] = M''_X(0) = \frac{3}{7} + \frac{1}{7} + \frac{9}{7} = \frac{13}{7}$$
   $$\text{Var}(X) = \frac{13}{7} - \left(\frac{1}{7}\right)^2 = \frac{91 - 1}{49} = \mathbf{\frac{90}{49}} \approx 1.8367$$
**Concepts:** [[06. Moments and Moment Generating Functions]], [[04. Expectation and Variance]]

### Question 3

#### (a) Two socks are selected at random and are removed in succession from a drawer containing five brown socks and three green socks. If random variable $W$ represents the number of brown socks selected then find the probability distribution of $W$ and $\text{E}(W)$.
**Solution:**
Total socks = $5 + 3 = 8$. Number of drawn socks $n = 2$.
$W$ can take values in $\{0, 1, 2\}$. Since the draws are in succession without replacement:
* **$\text{P}(W = 0)$ (both green):**
  $$\text{P}(W = 0) = \frac{\binom{3}{2}}{\binom{8}{2}} = \mathbf{\frac{3}{28}}$$
* **$\text{P}(W = 1)$ (one brown, one green):**
  $$\text{P}(W = 1) = \frac{\binom{5}{1}\binom{3}{1}}{\binom{8}{2}} = \mathbf{\frac{15}{28}}$$
* **$\text{P}(W = 2)$ (both brown):**
  $$\text{P}(W = 2) = \frac{\binom{5}{2}}{\binom{8}{2}} = \mathbf{\frac{10}{28}}$$
*(Sum of probabilities is $\frac{3+15+10}{28} = 1$).*

* **Calculate Expectation $\text{E}[W]$:**
  $$\text{E}[W] = 0\left(\frac{3}{28}\right) + 1\left(\frac{15}{28}\right) + 2\left(\frac{10}{28}\right) = \frac{35}{28} = \mathbf{\frac{5}{4}} = 1.25$$
**Concepts:** [[02. Discrete Random Variables]], [[04. Expectation and Variance]]

#### (b) The amount of bread (in hundreds of pounds) $x$ that a certain bakery is able to sell in a day is found to be a random phenomenon with a probability density function given by:
$$f(x) = \begin{cases} kx, & 0 \le x < 5 \\ k(10 - x), & 5 \le x < 10 \\ 0, & \text{otherwise} \end{cases}$$
##### i. Find the value of $k$ such that $f(x)$ is a probability density function.
##### ii. What is the probability that the number of pounds of bread that will be sold tomorrow is: (i) more than $500$ pounds (ii) less than $500$ pounds and (iii) between $250$ and $750$ pounds?

**Solution:**

##### i. Find $k$:
$$\int_{0}^{10} f(x) \, dx = 1 \implies \int_0^5 kx \, dx + \int_5^{10} k(10-x) \, dx = 1$$
$$k \left[ \frac{x^2}{2} \right]_0^5 + k \left[ 10x - \frac{x^2}{2} \right]_5^{10} = 1$$
$$k\left(\frac{25}{2}\right) + k\left((100 - 50) - (50 - 12.5)\right) = 1 \implies 12.5k + 12.5k = 1 \implies 25k = 1 \implies \mathbf{k = \frac{1}{25}}$$

##### ii. Compute probabilities (Note: $x$ is in hundreds of pounds, so 500 lbs corresponds to $x=5$):
* **(i) More than 500 pounds ($\text{P}(X > 5)$):**
  $$\text{P}(X > 5) = \int_5^{10} \frac{10-x}{25} \, dx = \frac{1}{25} \left[ 10x - \frac{x^2}{2} \right]_5^{10} = \frac{12.5}{25} = \mathbf{0.5}$$
* **(ii) Less than 500 pounds ($\text{P}(X < 5)$):**
  $$\text{P}(X < 5) = 1 - \text{P}(X > 5) = \mathbf{0.5}$$
* **(iii) Between 250 and 750 pounds ($\text{P}(2.5 < X < 7.5)$):**
  Using the symmetry of $f(x)$ around $x=5$:
  $$\text{P}(2.5 < X < 7.5) = 2 \int_{2.5}^5 \frac{x}{25} \, dx = \frac{2}{25} \left[ \frac{x^2}{2} \right]_{2.5}^5 = \frac{1}{25} \left(25 - 6.25\right) = \frac{18.75}{25} = \mathbf{0.75}$$
**Concepts:** [[03. Continuous Random Variables]]

### Question 4

#### (a) Two percent of books bound at a certain bindery have defective bindings. Use Poisson distribution to determine the probability that $5$ of $400$ books bound by this bindery will have defective bindings.
**Solution:**
Here, $n = 400$ and $p = 0.02$.
The Poisson parameter is:
$$\lambda = np = 400 \times 0.02 = 8$$
We want to find $\text{P}(X = 5)$ using the Poisson PMF:
$$\text{P}(X = 5) = \frac{e^{-8} 8^5}{5!} = \frac{32768 e^{-8}}{120} \approx 273.067 \times 0.000335 \approx \mathbf{0.0916}$$
**Concepts:** [[02. Discrete Random Variables]], [[05. Limit Theorems and Inequalities]]

#### (b) Let $X \sim \exp(4)$. Then, find $\text{P}(X \ge 5)$ and $\text{P}(4 \le X \le 7)$.
**Solution:**
For $X \sim \text{Exponential}(\lambda=4)$, the PDF is $f(x) = 4e^{-4x}$ for $x \ge 0$.
The CDF is $F(x) = 1 - e^{-4x}$.
1. **Find $\text{P}(X \ge 5)$:**
   $$\text{P}(X \ge 5) = 1 - F(5) = e^{-4(5)} = \mathbf{e^{-20}} \approx 2.06 \times 10^{-9}$$
2. **Find $\text{P}(4 \le X \le 7)$:**
   $$\text{P}(4 \le X \le 7) = F(7) - F(4) = (1 - e^{-28}) - (1 - e^{-16}) = \mathbf{e^{-16} - e^{-28}} \approx 1.13 \times 10^{-7}$$
**Concepts:** [[03. Continuous Random Variables]]

### Question 5

#### (a) Three random variables $X, Y$ and $Z$ have means $\mu_X = 2, \mu_Y = -3, \mu_Z = 4$, variances $\sigma_X^2 = 1, \sigma_Y^2 = 5, \sigma_Z^2 = 2$ and $\text{Cov}(X, Y) = -2, \text{Cov}(Y, Z) = -1$ and $\text{Cov}(Z, X) = 1$. Find $\text{Cov}(U, V)$ where $U = 3X - Y + 2Z$ and $V = X - 3Y + Z$.
**Solution:**
Using the bilinearity property of covariance:
$$\text{Cov}(U, V) = \text{Cov}(3X - Y + 2Z, X - 3Y + Z)$$
$$\text{Cov}(U, V) = 3\text{Cov}(X, X) - 9\text{Cov}(X, Y) + 3\text{Cov}(X, Z) - \text{Cov}(Y, X) + 3\text{Cov}(Y, Y) - \text{Cov}(Y, Z) + 2\text{Cov}(Z, X) - 6\text{Cov}(Z, Y) + 2\text{Cov}(Z, Z)$$
Note that $\text{Cov}(W, W) = \text{Var}(W)$. Substitute the known values:
$$\text{Cov}(U, V) = 3(1) - 9(-2) + 3(1) - (-2) + 3(5) - (-1) + 2(1) - 6(-1) + 2(2)$$
$$\text{Cov}(U, V) = 3 + 18 + 3 + 2 + 15 + 1 + 2 + 6 + 4 = \mathbf{54}$$
**Concepts:** [[03. Covariance and Sums]]

#### (b) Find mean and variance of random variable $X$ whose probability density function is:
$$f(x) = \begin{cases} \frac{4}{\pi(1+x^2)}, & 0 < x < 1 \\ 0, & \text{otherwise} \end{cases}$$
**Solution:**
1. **Find Mean $\text{E}[X]$:**
   $$\text{E}[X] = \int_{0}^{1} x f(x) \, dx = \int_0^1 \frac{4x}{\pi(1+x^2)} \, dx$$
   Let $u = 1 + x^2 \implies du = 2x \, dx$:
   $$\text{E}[X] = \frac{2}{\pi} \int_1^2 \frac{du}{u} = \frac{2}{\pi} [\ln(u)]_1^2 = \mathbf{\frac{2\ln(2)}{\pi}} \approx 0.4413$$

2. **Find Variance $\text{Var}(X) = \text{E}[X^2] - (\text{E}[X])^2$:**
   $$\text{E}[X^2] = \int_0^1 x^2 f(x) \, dx = \frac{4}{\pi} \int_0^1 \frac{x^2}{1+x^2} \, dx = \frac{4}{\pi} \int_0^1 \left(1 - \frac{1}{1+x^2}\right) \, dx$$
   $$\text{E}[X^2] = \frac{4}{\pi} \left[ x - \arctan(x) \right]_0^1 = \frac{4}{\pi} \left(1 - \frac{\pi}{4}\right) = \mathbf{\frac{4}{\pi} - 1} \approx 0.2732$$
   $$\text{Var}(X) = \left(\frac{4}{\pi} - 1\right) - \left(\frac{2\ln(2)}{\pi}\right)^2 = \mathbf{\frac{4}{\pi} - 1 - \frac{4\ln^2(2)}{\pi^2}} \approx 0.0785$$
**Concepts:** [[03. Continuous Random Variables]], [[04. Expectation and Variance]]

### Question 6

#### (a) An item is produced in large numbers. The machine is known to produce $5\%$ defectives. A quality control inspector is examining the items at random. What is the probability that at least $4$ items are to be examined in order to get $2$ defectives?
**Solution:**
This follows a Negative Binomial distribution where we wait for $r = 2$ successes (defectives), and the success probability is $p = 0.05$.
Let $Y$ represent the number of items examined.
We want to find $\text{P}(Y \ge 4) = 1 - \text{P}(Y < 4) = 1 - \text{P}(Y = 2) - \text{P}(Y = 3)$.
* **$\text{P}(Y = 2)$ (getting both defectives on first two trials):**
  $$\text{P}(Y = 2) = p^2 = (0.05)^2 = 0.0025$$
* **$\text{P}(Y = 3)$ (getting the 2nd defective on 3rd trial):**
  $$\text{P}(Y = 3) = \binom{3-1}{2-1} p^2 (1-p)^{3-2} = 2 (0.05)^2 (0.95) = 0.00475$$

$$\text{P}(Y \ge 4) = 1 - (0.0025 + 0.00475) = 1 - 0.00725 = \mathbf{0.99275}$$
**Concepts:** [[02. Discrete Random Variables]]

#### (b) If the probability is $0.75$ that an applicant for a driver's license will pass the road test on any given trial, what is the probability that an applicant will finally pass the test on the fourth try?
**Solution:**
This follows a Geometric distribution where success is passing ($p = 0.75$, $q = 0.25$).
We want to find the probability of success on the $k = 4$-th trial:
$$\text{P}(X = 4) = q^3 p = (0.25)^3 (0.75) = \frac{1}{64} \cdot \frac{3}{4} = \mathbf{\frac{3}{256}} \approx 0.0117$$

### Question 7

#### (a) $X$ has discrete uniform distribution, $f(x) = 1/k$, for $x = 1, 2, \dots, k$. Show that the moment generating function is given by:
$$\text{M}_X(t) = \text{E}(e^{tX}) = \frac{e^t(1-e^{kt})}{k(1-e^t)}$$
#### Also find $\text{M}'_X(0)$, $\text{M}''_X(0)$ and hence compute mean and variance of random variable $X$.
**Solution:**
1. **Show MGF:**
   $$M_X(t) = \text{E}[e^{tX}] = \sum_{x=1}^k e^{tx} \cdot \frac{1}{k} = \frac{1}{k} \sum_{x=1}^k (e^t)^x$$
   This is a finite geometric series with first term $a = e^t$, ratio $r = e^t$, and $k$ terms:
   $$M_X(t) = \frac{1}{k} \cdot \frac{e^t(1 - (e^t)^k)}{1 - e^t} = \mathbf{\frac{e^t(1-e^{kt})}{k(1-e^t)}} \quad \blacksquare$$

2. **Compute Mean and Variance:**
   Differentiating this MGF is algebraically intensive. A simpler way is to find the moments directly:
   * **Mean ($\text{E}[X]$):**
     $$\text{E}[X] = \sum_{x=1}^k x \cdot \frac{1}{k} = \frac{1}{k} \sum_{x=1}^k x = \frac{1}{k} \left(\frac{k(k+1)}{2}\right) = \mathbf{\frac{k+1}{2}}$$
   * **Variance ($\text{Var}(X)$):**
     $$\text{E}[X^2] = \sum_{x=1}^k x^2 \cdot \frac{1}{k} = \frac{1}{k} \left(\frac{k(k+1)(2k+1)}{6}\right) = \frac{(k+1)(2k+1)}{6}$$
     $$\text{Var}(X) = \text{E}[X^2] - (\text{E}[X])^2 = \frac{2k^2 + 3k + 1}{6} - \frac{k^2 + 2k + 1}{4} = \frac{4k^2 + 6k + 2 - (3k^2 + 6k + 3)}{12} = \mathbf{\frac{k^2 - 1}{12}}$$
**Concepts:** [[06. Moments and Moment Generating Functions]], [[02. Discrete Random Variables]], [[04. Expectation and Variance]]

#### (b) Let $X_1$ and $X_2$ be two independent Poisson variates with parameters $\lambda_1$ and $\lambda_2$ respectively. Prove that $X_1 + X_2$ follows Poisson distribution with parameter $\lambda_1 + \lambda_2$. Does $X_1 - X_2$ also follow Poisson distribution? Give reasons.
**Solution:**
1. **Prove sum is Poisson:**
   Let $M_{X_i}(t) = e^{\lambda_i(e^t-1)}$ be the MGF of $X_i$.
   Since $X_1$ and $X_2$ are independent:
   $$M_{X_1+X_2}(t) = M_{X_1}(t) \cdot M_{X_2}(t) = e^{\lambda_1(e^t-1)} e^{\lambda_2(e^t-1)} = e^{(\lambda_1+\lambda_2)(e^t-1)}$$
   This is exactly the MGF of a Poisson distribution with parameter $\lambda_1 + \lambda_2$. By the uniqueness theorem of MGFs, $X_1 + X_2 \sim \text{Poisson}(\lambda_1 + \lambda_2)$.

2. **Does $X_1 - X_2$ follow Poisson?**
   **No.** A Poisson random variable can only take non-negative integer values $\{0, 1, 2, \dots\}$. However, the difference $X_1 - X_2$ can take negative values (for example, if $X_1 = 0$ and $X_2 = 1$, then $X_1 - X_2 = -1$). Therefore, its range is the set of all integers $\mathbb{Z}$, which violates the Poisson definition.
**Concepts:** [[06. Moments and Moment Generating Functions]], [[02. Discrete Random Variables]]

#### (b) If $X$ is binomially distributed with parameters $n$ and $p$, then what is the distribution of $Y = n - X$?
**Solution:**
Since $X$ is Binomial, its MGF is $M_X(t) = (q + p e^t)^n$ (where $q = 1-p$).
Find the MGF of $Y = n - X$:
$$M_Y(t) = \text{E}[e^{t(n-X)}] = e^{nt} M_X(-t) = e^{nt} (q + p e^{-t})^n = (q e^t + p)^n = (p + q e^t)^n$$
This is the MGF of a Binomial distribution with parameters $n$ and $q = 1-p$.
By uniqueness of MGF, $Y \sim \mathbf{\text{Binomial}(n, 1-p)}$.
**Concepts:** [[06. Moments and Moment Generating Functions]], [[02. Discrete Random Variables]]

### Question 9

#### (a) Subway trains on a certain line run every half hour between midnight and six in the morning. What is the probability that a man entering the station at a random time during this period will have to wait at least twenty minutes?
**Solution:**
Let $T$ be the waiting time in minutes. Since trains run every 30 minutes, $T$ follows a Uniform distribution:
$$T \sim \text{Uniform}(0, 30)$$
The PDF of $T$ is $f(t) = \frac{1}{30}$ for $0 \le t \le 30$.
$$\text{P}(T \ge 20) = \int_{20}^{30} \frac{1}{30} \, dt = \left[ \frac{t}{30} \right]_{20}^{30} = \frac{30 - 20}{30} = \mathbf{\frac{1}{3}} \approx 0.3333$$
**Concepts:** [[03. Continuous Random Variables]]

---

#### (b) Obtain moment generating function of the Geometric distribution. Also, find its mean and variance.
**Solution:**
Let $X \sim \text{Geometric}(p)$, where $p(x) = q^{x-1}p$ for $x = 1, 2, \dots$
1. **Find MGF:**
   $$M_X(t) = \text{E}[e^{tX}] = \sum_{x=1}^{\infty} e^{tx} q^{x-1}p = p e^t \sum_{x=1}^{\infty} (q e^t)^{x-1}$$
   This is an infinite geometric series with first term 1 and common ratio $q e^t$. It converges if $q e^t < 1$ ($t < -\ln(q)$):
   $$M_X(t) = \mathbf{\frac{p e^t}{1 - q e^t}}$$
2. **Find Mean and Variance:**
   See derivations in [[04. Expectation and Variance#C. Geometric Distribution (X sim Geometric(p))]] and MGF derivations in [[06. Moments and Moment Generating Functions#5. MGF Table for Core Distributions]].
   $$\text{Mean} = \mathbf{\frac{1}{p}}, \quad \text{Variance} = \mathbf{\frac{1-p}{p^2}}$$

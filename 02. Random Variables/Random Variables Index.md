# Unit 2: Random Variables

This unit covers random variables, their distributions, expectation, variance, key discrete and continuous distributions, inequalities, and core limit theorems.

## 📝 Detailed Study Guides

1. [[01. Introduction to Random Variables and CDF]] — Definition of random variables, the indicator variable, the collector's problem, and CDF properties.
2. [[02. Discrete Random Variables]] — Discrete distributions (Bernoulli, Binomial, Geometric, Poisson) and the Poisson approximation derivation.
3. [[03. Continuous Random Variables]] — Continuous distributions (Uniform, Exponential, Normal), normal variable linear transformations, and the standard normal. *(Excludes Gamma distribution).*
4. [[04. Expectation and Variance]] — Definition of expected values, detailed derivations for all key distributions, expected value of a function, and variance proofs.
5. [[05. Limit Theorems and Inequalities]] — Analytical proofs for Markov's and Chebyshev's inequalities, Law of Large Numbers, CLT MGF proof, and normal approximations.

# 04. Conditional Expectation and Distributions (Section 3.1, 3.2, 3.3)

In Unit 1, we learned about conditional probability for events: $P(E|F)$. Now, we extend this concept to random variables. What is the distribution of a variable $X$, given that we have observed the exact value of another variable $Y = y$?

---

## 1. The Core Analogy: "Slicing the Probability Map"

> 💡 **Mental Model**: Imagine a 3D clay model of a hill representing the joint PDF $f(x, y)$. 
> - If we are told that $Y = y$ exactly, we take a knife and **slice the hill** right along the line $Y = y$.
> - This slice gives us a 2D cross-section. However, the area under this cross-section is no longer equal to 1. 
> - To make it a valid probability distribution, we **re-scale** the slice by dividing it by the total height/area of the slice (which is the marginal density $f_Y(y)$).

```
    Joint PDF hill:        Slice along Y=y:       Re-scaled to 100% area:
        .---.                   .---.                    .---.
       /     \                 /     \                  /     \
      /       \               /       \                /       \
     /         \             /─────────\              /         \
                             (Cut Surface)            (Valid PDF)
```

## 2. The Discrete Case: Conditional PMF & Expectation

If $X$ and $Y$ are discrete:

> **Definition**: The **conditional probability mass function** of $X$ given $Y=y$ is:
> $$p_{X|Y}(x|y) = P(X=x | Y=y) = \frac{P(X=x, Y=y)}{P(Y=y)} = \frac{p(x, y)}{p_Y(y)}$$
> where $p_Y(y) > 0$.

### Conditional Expectation
The **conditional expectation** of $X$ given $Y=y$ is simply the expected value calculated using the conditional PMF:
$$E[X|Y=y] = \sum_x x \, p_{X|Y}(x|y)$$

---

## 3. The Continuous Case: Conditional PDF & Expectation

If $X$ and $Y$ are continuous:

> **Definition**: The **conditional probability density function** of $X$ given $Y=y$ is:
> $$f_{X|Y}(x|y) = \frac{f(x, y)}{f_Y(y)}$$
> where $f_Y(y) > 0$.

### Conditional Expectation
The **conditional expectation** of $X$ given $Y=y$ is:
$$E[X|Y=y] = \int_{-\infty}^\infty x \, f_{X|Y}(x|y) \, dx$$

## 4. Walkthrough Example: Discrete Conditioning (Example 3.2)

* **Problem**: Let $X_1$ and $X_2$ be independent Binomial variables with parameters $(n_1, p)$ and $(n_2, p)$. Find the conditional PMF of $X_1$ given that $X_1 + X_2 = m$.
* **Step-by-step Walkthrough**:
  1. We want to find $P(X_1 = k | X_1 + X_2 = m)$. By definition:
     $$P(X_1 = k | X_1 + X_2 = m) = \frac{P(X_1 = k, X_1 + X_2 = m)}{P(X_1 + X_2 = m)}$$
  2. The event $\{X_1 = k \text{ and } X_1 + X_2 = m\}$ is identical to the event $\{X_1 = k \text{ and } X_2 = m - k\}$.
  3. Since $X_1$ and $X_2$ are independent:
     $$P(X_1 = k | X_1 + X_2 = m) = \frac{P(X_1 = k) P(X_2 = m-k)}{P(X_1 + X_2 = m)}$$
  4. Recall that the sum of independent Binomials with the same parameter $p$ is itself Binomial: $X_1 + X_2 \sim \text{Binomial}(n_1 + n_2, p)$.
  5. Substitute the PMFs (with $q = 1-p$):
     $$P(X_1 = k | X_1 + X_2 = m) = \frac{\binom{n_1}{k} p^k q^{n_1-k} \binom{n_2}{m-k} p^{m-k} q^{n_2 - m + k}}{\binom{n_1+n_2}{m} p^m q^{n_1+n_2-m}}$$
  6. Simplify the $p$ and $q$ terms:
     - Numerator $p^k \cdot p^{m-k} = p^m$. This cancels with the denominator $p^m$.
     - Numerator $q^{n_1-k} \cdot q^{n_2-m+k} = q^{n_1+n_2-m}$. This cancels with the denominator.
  7. The remaining terms are:
     $$P(X_1 = k | X_1 + X_2 = m) = \frac{\binom{n_1}{k} \binom{n_2}{m-k}}{\binom{n_1+n_2}{m}}$$
* **Conclusion**: This is exactly the PMF of a **Hypergeometric distribution**! 
* *Intuition*: Imagine you have $n_1$ blue balls and $n_2$ red balls. You choose a total sample of $m$ balls. The probability that your sample contains exactly $k$ blue balls is given by this Hypergeometric formula.

## 5. Walkthrough Example: Continuous Integration (Example 3.6)

* **Problem**: The joint density of $X$ and $Y$ is:
  $$f(x, y) = 6xy(2 - x - y), \quad 0 < x < 1, 0 < y < 1$$
  Compute the conditional expectation $E[X|Y=y]$.
* **Step-by-step Walkthrough**:
  1. **Find the marginal density of $Y$, $f_Y(y)$**:
     We integrate out $x$ over the interval $[0, 1]$:
     $$\begin{aligned}
     fY(y) &= \int_0^1 6xy(2 - x - y) \, dx = 6y \int_0^1 (2x - x^2 - yx) \, dx \\
     &= 6y \left[ x^2 - \frac{x^3}{3} - \frac{yx^2}{2} \right]_0^1 = 6y \left( 1 - \frac{1}{3} - \frac{y}{2} \right) \\
     &= 6y \left( \frac{2}{3} - \frac{y}{2} \right) = y(4 - 3y)
     \end{aligned}$$
  2. **Find the conditional density $f_{X|Y}(x|y)$**:
     $$f_{X|Y}(x|y) = \frac{f(x, y)}{f_Y(y)} = \frac{6xy(2 - x - y)}{y(4 - 3y)} = \frac{6x(2 - x - y)}{4 - 3y}$$
  3. **Calculate the expectation $E[X|Y=y]$**:
     $$\begin{aligned}
     E[X|Y=y] &= \int_0^1 x f_{X|Y}(x|y) \, dx = \int_0^1 x \left[ \frac{6x(2 - x - y)}{4 - 3y} \right] dx \\
     &= \frac{6}{4 - 3y} \int_0^1 (2x^2 - x^3 - yx^2) \, dx \\
     &= \frac{6}{4 - 3y} \left[ \frac{2x^3}{3} - \frac{x^4}{4} - \frac{yx^3}{3} \right]_0^1 = \frac{6}{4 - 3y} \left( \frac{2}{3} - \frac{1}{4} - \frac{y}{3} \right) \\
     &= \frac{6}{4 - 3y} \left( \frac{5 - 4y}{12} \right) = \frac{5 - 4y}{2(4 - 3y)} = \frac{5 - 4y}{8 - 6y} \quad \blacksquare
     \end{aligned}$$
